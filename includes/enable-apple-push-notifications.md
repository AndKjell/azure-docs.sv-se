
* [Registrieren Sie eine App-ID für Ihre app](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/MaintainingProfiles/MaintainingProfiles.html#//apple_ref/doc/uid/TP40012582-CH30-SW991). Erstellen Sie eine explizite App-ID (keine Platzhalter-App-ID) und **-Bündel-ID**, die genaue **Bundle-ID** die sich in Ihrem Xcode-schnellstartprojekt. Es ist auch wichtig, dass Sie überprüfen die **Pushbenachrichtigungen** Option. 

* Als Nächstes [Konfigurieren von Pushbenachrichtigungen](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/AddingCapabilities/AddingCapabilities.html#//apple_ref/doc/uid/TP40012582-CH26-SW6). Sie können ein SSL-Zertifikat für "Entwicklung" oder "Vertrieb" erstellen (denken Sie daran, später im Azure-Portal die entsprechende Option auszuwählen).
