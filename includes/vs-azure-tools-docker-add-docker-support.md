    ![Lägg till stöd för Docker-snabbmeny](media/vs-azure-tools-docker-add-docker-support/docker-support-context-menu.png)
2. Att lägga till Docker-stöd i en ASP.NET Core resulterar-webbprojekt i att lägga till flera Docker-relaterade filer som läggs till i projektet, inklusive Dockerfile, distribution av Windows PowerShell-skript och filer för Docker-egenskapen. 
   
    ![Dockerfiler som läggs till i projektet](media/vs-azure-tools-docker-add-docker-support/docker-files-added.png)
