Kontrollera att du använder katalogen som innehåller din Azure AD B2C-klient genom att växla till den i det övre högra hörnet i Azure-portalen. Välj din prenumerationsinformation och välj därefter **Växla katalog**. 

![Växla till Azure AD B2C-klientorganisationen](./media/active-directory-b2c-switch-b2c-tenant/switch-directories.png)

Välj den katalog som innehåller din klient.
