För lokal utveckling i Azure Functions version2.x paketet används för att [registrera bindning tillägget](../articles/azure-functions/functions-triggers-bindings.md#local-development-azure-functions-core-tools). Klassen biblioteket utveckling i funktioner för C# 1.x och 2.x, stöd för bindningen läggs till av [installera paketet](../articles/azure-functions/functions-triggers-bindings.md#local-c-development-using-visual-studio-or-vs-code).

Observera att källkoden följer WebJobs SDK version numrering: WebJobs SDK version 2.x motsvarar Azure Functions 1.x, så funktioner 1.x koden finns i grenen v2.x i databasen. Huvud- och dev filialer innehåller koden för Web jobb 3.x, vilket är samma som fungerar 2.x.
