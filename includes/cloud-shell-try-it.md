
## <a name="open-azure-cloud-shell"></a>Öppna Azure-molnet Shell

Azure Cloud-gränssnittet är ett kostnadsfritt, interaktiva gränssnitt som du kan använda för att köra stegen i den här artikeln. Vanliga verktyg för Azure förinstallerat och konfigureras i molnet Shell som du kan använda med ditt konto. Bara väljer den **kopiera** för att kopiera koden, klistra in den i molnet Shell och tryck sedan på RETUR för att köra den. Det finns några olika sätt att öppna molnet Shell:

|  |   |
|-----------------------------------------------|---|
| Välj **prova** i det övre högra hörnet av ett kodblock. | ![Cloud Shell i den här artikeln](./media/cloud-shell-try-it/cli-try-it.png) |
| Öppna Cloud Shell i din webbläsare. | [![https://shell.azure.com/bash](./media/cloud-shell-try-it/launchcloudshell.png)](https://shell.azure.com/bash) |
| Välj den **moln Shell** knappen på menyn i det övre högra hörnet på de [Azure-portalen](https://portal.azure.com). |    ![Cloud Shell i portalen](./media/cloud-shell-try-it/cloud-shell-menu.png) |
|  |  |
