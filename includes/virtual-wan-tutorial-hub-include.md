En ny hubb innehåller gatewayen, som tar 30 minuter att skapa. När hubben har skapats debiteras du för hubben, även om du inte kopplar några platser. 
1. På WAN-sidan går du till **Virtuell WAN-arkitektur** och klickar på **Hubbar** för att sidan Hubbar.
2. På sidan Hubbar klickar du på **+ Ny hubb** för att öppna sidan **Hubbar**.
3. På sidan Hubbar fyller du i följande fält:

  * Plats
  * Namn
  * Privat adressutrymme för hubb
4. Klicka på **Bekräfta** för att skapa hubben. Klicka på **Uppdatera** för att visa hubben på sidan Hubbar.
