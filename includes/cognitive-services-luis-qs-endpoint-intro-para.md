I den här snabbstarten använder du en tillgänglig offentliga LUIS-app för att fastställa användarens avsikt från konversationsanpassade text. Skicka användarens avsikt som text till offentlig app HTTP-slutpunkt för förutsägelse. Vid slutpunkten gäller LUIS offentliga appens modellen för att analysera texter med naturligt språk för vilket innebär att, avgör övergripande avsikten och extrahera data som är relevanta för ämne programdomän. 

Den här snabbstarten använder slutpunkten REST API. Mer information finns i den [endpoint API-dokumentation](https://westus.dev.cognitive.microsoft.com/docs/services/5819c76f40a6350ce09de1ac/operations/5819c77140a63516d81aee78).

I den här artikeln behöver du en kostnadsfri [LUIS](http://www.luis.ai) konto.
