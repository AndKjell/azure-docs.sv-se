

Pushbenachrichtigungen werden normalerweise in einem Back-End-Dienst wie z. B. Mobile Services oder ASP.NET mit einer kompatiblen Bibliothek gesendet. Sie können Benachrichtigungsmeldungen zudem direkt über die REST-API senden, wenn für Ihr Back-End keine Bibliothek verfügbar ist. 

Es folgen einige andere Lernprogramme, die Informationen zum Senden von Benachrichtigungen enthalten:

- Azure Mobile Services: Ein Beispiel für das Senden von Benachrichtigungen von einem mit Notification Hubs integrierten Azure Mobile Services-Back-End finden Sie unter [Erste Schritte mit Pushbenachrichtigungen in Mobile Services].  
- ASP.NET: [Verwenden von Notification Hubs für Pushbenachrichtigungen an Benutzer].
- Azure Notification Hub Java SDK: finden Sie unter [mit Notification Hubs von Java](../articles/notification-hubs/notification-hubs-java-backend-how-to.md) zum Senden von Benachrichtigungen über Java. Dies wurde für die Android-Entwicklung in Eclipse getestet.
- PHP: [mit Notification Hubs von PHP](../articles/notification-hubs/notification-hubs-php-backend-how-to.md).


Im nächsten Abschnitt des Lernprogramms erfahren Sie, wie mithilfe der [Notification Hub-REST-Schnittstelle](http://msdn.microsoft.com/library/windowsazure/dn223264.aspx) die Nachricht direkt in Ihrer app zu senden. Die von einem Gerät gesendeten Benachrichtigungen werden auf allen registrierten Geräten empfangen.
