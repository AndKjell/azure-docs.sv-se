Ressource|Standardlimit|Maximales Limit
---|---|---
[Virtuelle Computer](../articles/virtual-machines/virtual-machines-about.md) pro Cloud-Dienst<sup>1</sup>|50|50
Eingabeendpunkte pro Clouddienst<sup>2</sup>|150|150

<sup>1</sup>Die in der Dienstverwaltung (und nicht im Ressourcen-Manager) erstellten virtuellen Computer werden automatisch in einem Clouddienst gespeichert. Für den Lastenausgleich und die Verfügbarkeit können Sie diesem Clouddienst weitere virtuelle Computer hinzufügen. Finden Sie unter  [wie virtuelle Maschinen mit einem virtuellen Netzwerk verbinden oder Cloud-Dienst](../articles/virtual-machines/cloud-services-connect-virtual-machine.md).

<sup>2</sup>Eingabeendpunkte ermöglichen die Kommunikation mit einem virtuellen Computer von außerhalb des Clouddiensts des virtuellen Computers. Virtuelle Computer im gleichen Clouddienst oder virtuellen Netzwerk können automatisch miteinander kommunizieren. Finden Sie unter [Einrichten von Endpunkten für einen virtuellen Computer](../articles/virtual-machines/virtual-machines-set-up-endpoints.md).
