> [!NOTE]
> **Anpassade principer finns i förhandsversion.**

> Anpassade principer utformade främst för att lösa avancerade scenarier. För de flesta fall rekommenderar vi att du använder Azure Active Directory B2C [inbyggda principer](..\articles\active-directory-b2c\active-directory-b2c-reference-policies.md). Inbyggda principerna är lättare att ställa in för din konfiguration.
