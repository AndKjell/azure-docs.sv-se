Für Azure Site Recovery gelten die folgenden Beschränkungen.


| LIMIT IDENTIFIER| STANDARD-LIMIT|
|---|---|---|---|

| Anzahl der Wiederherstellungspläne pro Azure vault| Keine limit|
| Anzahl der Server pro Schutzgruppe Gruppe | Keine limit|
