Skapa och hantera kluster molnbaserade högpresterande datorbearbetning (HPC) genom att utnyttja [Microsoft HPC Pack](https://technet.microsoft.com/library/jj899572.aspx) och Azure beräkning och infrastruktur-tjänster. HPC Pack tillgängliga gratis hämta bygger på Microsoft Azure och Windows Server-teknik och stöder ett stort antal HPC-arbetsbelastningar.

Fler HPC-alternativ i Azure, se [tekniska resurser för batch- och högpresterande datorbearbetning](../articles/batch/big-compute-resources.md).
