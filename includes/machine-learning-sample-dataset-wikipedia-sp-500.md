Daten stammen aus Wikipedia (<a href="http://www.wikipedia.org/">http://www.wikipedia.org/</a>) basieren auf Artikeln der einzelnen S & P 500-Unternehmen, die als XML-Daten gespeichert.<p> </p>Das Dataset wurde vor dem Hochladen in Azure ML Studio wie folgt verarbeitet:<ul><li>Extrahieren der Textinhalte für die einzelnen Unternehmen</li><li>Entfernen der Wiki-Formatierung</li><li>Entfernen aller nicht alphanumerischen Zeichen</li><li>Konvertieren sämtlicher Texte in Kleinbuchstaben</li><li>Bekannte Firmenkategorien wurden hinzugefügt.</li></ul><p> </p>Beachten Sie, dass für einige Unternehmen keine Artikel gefunden werden konnten, daher ist die Anzahl der Datensätze kleiner als 500.
