# <a name="get-started-with-the-excel-connector"></a>Komma igång med Excel-koppling
Det finns ingen koppling i Excel i Logic Apps. 

## <a name="to-use-excel-data"></a>Använda Excel-data
Du kan lagra Excel-data som en fil med kommaavgränsade värden (CSV) i en mapp, till exempel [OneDrive](connectors-create-api-onedrive.md). Du kan också använda den här CSV-filen med den [2D-filen connector](../logic-apps/logic-apps-enterprise-integration-flatfile.md).

## <a name="connector-specific-details"></a>Connector-specifik information

Visa alla utlösare och åtgärder som definierats i swagger och även se några gränser i den [connector information](/connectors/excel/).
