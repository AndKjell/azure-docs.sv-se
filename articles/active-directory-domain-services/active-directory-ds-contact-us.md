# <a name="azure-ad-domain-services---contact-us"></a>Azure AD Domain Services - kontaktar du oss
## <a name="contact-the-product-team"></a>Kontakta produktteamet
Om du har problem med din hanterade domän kontrollerar du om stegen som beskrivs i den [felsökningsguiden](active-directory-ds-troubleshooting.md) lösa problemet. Om du fortfarande har problem, gärna kontakta oss.

Du kan skicka e-post på: [Azure AD Domain Services Feedback](mailto:aaddsfb@microsoft.com).

Se till att du inkluderar följande, så kan vi undersöka problemet.

* Den **klient-ID eller katalogen ID** för Azure AD-katalogen. Klient-ID är det GUID som du ser i fältet katalog-ID på den [egenskapssidan för din katalog](https://ms.portal.azure.com/#blade/Microsoft_AAD_IAM/ActiveDirectoryMenuBlade/Properties)
* Den **DNS-domännamn** du har konfigurerat för din hanterade domän AAD Domain Services.

## <a name="provide-feedback"></a>Skicka feedback
Vi uppskattar din feedback om Azure AD Domain Services via den  **[Azure Active Directory User Voice kanal](https://feedback.azure.com/forums/169401-azure-active-directory/)**.

Se till att du pre-väntande frågan eller feedback med orden **'AADDS'**, att nå oss.
