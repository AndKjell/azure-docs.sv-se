#<a name="azure-government-documentation"></a>Dokumentation om Azure Government

Microsoft Azure Government erbjuder en molnplattform byggd på de grundläggande principerna säkerhet, sekretess och kontroll, kompatibilitet och transparens. Entiteter inom offentlig sektor får en fysiskt isolerad instans av Microsoft Azure som använder sig av de säkerhets- och kompatibilitetstjänster i världsklass som den amerikanska regeringen förlitar sig på för alla system och program som skapas på dess arkitektur.
