# <a name="updates-to-the-sign-in-experience-for-online-services"></a>Uppdateringar till inloggningen för onlinetjänster

Vi uppdaterar vår erfarenhet för Microsofts onlinetjänster som Office 365 och Azure. Detta är endast en visuell ändring, Lägg märke till att skärmarna verka mindre plottrigt och enklare, vilket gör det enklare för dig att fokusera på vad du behöver göra – logga in.

## <a name="current-ui-versus-new-ui"></a>Aktuella UI jämfört med nya Användargränssnittet

Checka ut befintliga Användargränssnittet jämfört med det nya Användargränssnittet. Det nya Användargränssnittet är mer effektiv och fokus med ett uppdaterat känsla och utseende.

![gamla Microsoft inloggning](media/microsoft-sign-in-experience-old.jpg)

![nya Microsoft-inloggningsupplevelsen](media/microsoft-sign-in-experience-new.jpg)

## <a name="when-will-the-new-ui-appear"></a>När visas det nya Användargränssnittet?

Här är en tidslinje på när du ser den här ändringen ska inträffa:

| Tidsram | Ändra |
|-----------|--------|
| Tidig kan | Du börjar att visa en meddelandebanderoll på din befintliga inloggningssidan. Härifrån kan du lär dig mer om och förbereda för att ändringarna. |
| Halva kan   | Meddelande banderoll ändringarna inkludera en länk om du vill förhandsgranska uppdaterade Användargränssnittet. När i preview kan du fortfarande använda en länk växla tillbaka till den befintliga miljön. |
| Mid juni  | Uppdaterade inloggning Gränssnittet är tillgängligt och alla standardvärden för nya användargränssnitt. |

## <a name="related-info"></a>Närliggande information

Om du är administratör kan hittar du mer information om dessa uppdateringar baserat på produkt.

- **Office 365**. Mer information om uppdateringarna till [meddelandecenter för Office 365](https://portal.office.com/adminportal/home#/MessageCenter?id=MC133397&MCLinkSource=DigestMail). Du måste logga in med hjälp av admin-autentisering till den här webbplatsen.

- **Andra onlinetjänster**. Få mer information om uppdateringarna genom att läsa detta [blogginlägget](https://cloudblogs.microsoft.com/enterprisemobility/2018/04/04/upcoming-improvements-to-the-azure-ad-sign-in-experience/).
