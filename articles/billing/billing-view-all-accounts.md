# <a name="view-all-your-billing-accounts-in-azure-portal"></a>Visa din faktureringsadress konton i Azure-portalen  

Fakturering konton definiera dina Azure fakturering relationer. Du kan ha flera konton för fakturering. Du kan till exempel ha en Azure-prenumeration som du använder för personliga projekt. Du kan också ha åtkomst till Azure via din organisations Enterprise-avtal. Gå till [kostnaden Management + fakturering](https://portal.azure.com/#blade/Microsoft_Azure_Billing/BillingMenuBlade/Overview) i Azure portal för att visa din faktureringsadress konton.

Du kan för närvarande bara visa dina personliga och Enterprise-avtal fakturering konton. Som standard den [kostnaden Management + fakturering](https://portal.azure.com/#blade/Microsoft_Azure_Billing/BillingMenuBlade/Overview) visar ditt personliga faktureringskonto. Dock kan du gå till **fakturering konton** och ändra kontot.

För personliga fakturering konton som du kan hantera och visa alla ändringar i den [Azure-portalen](https://portal.azure.com). Men för Enterprise-avtal konton, kan du bara visa reserverade inköp, belastningsutjämning och månad hittills avgifter. Dessa funktioner för Enterprise-avtal finns i förhandsgranskningen och är bara tillgängligt om du är i ett Enterprise-administratörsrollen.

## <a name="change-billing-account-view"></a>Ändra fakturering konto 

1.  Logga in på [Azure-portalen](https://portal.azure.com).

2.  Välj vänster sida av portalen **kostnaden Management + fakturering**. 

3.  Som standard om du använder en, ser du ditt personliga faktureringskonto. Om du inte har en personlig faktureringskonto eller om du vill ändra till en Enterprise-avtal kontot väljer **fakturering konton**.

4.  Välj ett konto för att visa information. För Enterprise-avtal visas bara fakturering konton, där du har en Enterprise-administratör. Andra roller i Enterprise-avtal kommer att stödjas framöver.

    ![Skärmbild som visar listan över konton för fakturering](./media/billing-view-all-accounts/billing-list-of-accounts.png)
