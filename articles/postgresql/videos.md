# <a name="azure-database-for-postgresql-videos"></a>Azure-databas för PostgreSQL-videor
[Hur du kommer igång med den nya Azure-databasen för PostgreSQL-tjänsten](https://channel9.msdn.com/events/Build/2017/B8046)

<iframe src="https://channel9.msdn.com/Events/Build/2017/B8046/player" width="800" height="450" allowFullScreen frameBorder="0"></iframe>

I den här videon från 2017 Microsoft //Build konferens lära dig från två tidigare använda kunder hur de har använt Azure-databas för PostgreSQL-tjänsten för att förnya snabbare. Lär dig hur de migreras till tjänsten och diskutera nästa steg i sina programutveckling. Videon går igenom några av funktionerna som nyckel och beskriver hur du som en utvecklare kan migrera dina befintliga program eller utveckla nya program som använder den här hanterade PostgreSQL-tjänsten i Azure.
