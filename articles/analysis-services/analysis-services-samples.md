# <a name="azure-analysis-services-samples"></a>Azure Analysis Services-exempel
Använd följande exempel resurser för att du lär dig mer om och testa Analysis Services för din miljö.

## <a name="code-samples"></a>Kodexempel
Den [Analysis Services](https://github.com/Microsoft/Analysis-Services) databasen på GitHub innehåller kodexempel för öppen källkod och community-projekt. 

## <a name="tabular-model-project-and-database-samples"></a>Tabellmodell projekt och databasen prover
[Adventure Works för Analysis Services](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks-analysis-services) är de vanligaste tabellmodell exempelprojektet på GitHub. Du kan hämta VS-projektet eller en tabellmodell för slutförda exempeldatabas.


## <a name="sample-databases-on-github"></a>Exempeldatabaserna på GitHub
Dessa Exempeldatabaser på GitHub kan användas för att skapa och testa din egen modeller.

[Wide World Importers exempeldatabas](https://github.com/Microsoft/sql-server-samples/releases/tag/wide-world-importers-v1.0)

## <a name="adventure-works-internet-sales-sample-model-in-azure-portal"></a>Adventure Works Internet försäljning exempel modellen i Azure-portalen
Om du har en Analysis Services-server i Azure-portalen kan snabbt och skapa enkelt en exempel-modell. På din server översikt i Azure-portalen klickar du på **ny modell**, och klicka sedan på **Välj en datakälla**väljer **exempeldata**.
