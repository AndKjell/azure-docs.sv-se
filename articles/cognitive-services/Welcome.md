# <a name="what-is-cognitive-services"></a>Vad är Cognitive Services?

Microsofts är kognitiva (tidigare projekt Oxford) en uppsättning API: er, SDK: er och tjänster som är tillgänglig för utvecklare att göra deras program intelligent, spännande och kan identifieras. Microsoft kognitiva Services kan utökas med Microsofts växande portfölj av machine learning API: er och gör att utvecklare kan enkelt lägga till intelligent funktioner – till exempel känsla och video identifiering; ansiktskirurgi tal- och vision recognition; och tal och språk förstå – i sina program. Visionen är mer personliga databehandling upplevelser och förbättrad produktivitet genom system som allt kan se, höra, tala, förstå och även börja orsak.

Du hittar dokumentationen för varje produkt och deras motsvarande API-referenserna för den vänstra tabellen.

## <a name="getting-started-with-free-trials"></a>Komma igång med gratisutvärderingar
Registrera dig för försök tar bara ett e-postmeddelande och några få [enkla steg](https://azure.microsoft.com/try/cognitive-services/ "anmälan hjälpa") du behöver ett Account om du inte redan har ett. Du får ett unikt par nycklar för varje API som begärdes. Den andra är bara en snabbersättningsdisk. Ange dela inte hemliga nycklar med vem som helst. Försök har båda hastighetsbegränsning vad gäller transaktioner per sekund eller minut och gräns för en månad användning. En transaktion är bara ett API-anrop. Du kan uppgradera till betalnivåer att låsa upp begränsningarna.

## <a name="subscription-management"></a>Prenumerationshantering
När du har loggat in med Account du kommer att kunna komma åt [Mina prenumerationer] (https://www.microsoft.com/cognitive-services/en-us/subscriptions ”Mina prenumerationer”) så att de produkter som du använder, den kvot och möjligheten att lägga till ytterligare produkter i prenumerationen.

## <a name="upgrade-to-unlock-limits"></a>Uppgraderingen för att låsa upp gränser
Alla API: er har en kostnadsfri utvärderingsversion plan.  När betald erbjudanden blir tillgängliga för varje API kommer du till Azure-portalen att slutföra köpet.  Du kan hitta *köpa* länkar dina prenumerationer på sidan om du redan använder dem eller om du kan hoppa över utvärderingsversion helt och hållet och köpa med tillhandahålls länkar på [priser](https://www.microsoft.com/cognitive-services/en-us/pricing "priser").  Du måste skapa ett Azure-prenumerant konto med ett kreditkort och ett telefonnummer. Om du har särskilda krav eller för att kommunicera med försäljning, klickar du på ”Kontakta oss” längst upp på sidan med priser.

## <a name="regional-availability"></a>Regional tillgänglighet
API: er i kognitiva tjänster finns på en växande nätverk med Microsoft-hanterade datacenter. Du kan hitta regional tillgänglighet för varje API i [Azure-regionlistan](https://azure.microsoft.com/regions).
    
Söker efter en region stöds inte ännu? Berätta för oss genom att arkivera en funktionsbegäran på vår [UserVoice forum](https://cognitive.uservoice.com/).
