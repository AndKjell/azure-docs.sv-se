# <a name="bing-search-sdk"></a>Bing Search SDK
Bing enheten Sök API-exempel finns scenarier som:
1.  Sök efter enhet, till exempel Tom Cruise och få omfattande information.
2.  Hantera disambiguation av villkor för frågor med eventuellt flera avsikter.
3.  Sök efter en lokal enhet, till exempel en restaurang och få omfattande information runt den.
4.  Sök efter lokala företag, till exempel hotell och få omfattande information.
5.  Utlös en felaktig begäran och felhantering.

> [!NOTE] 
> Vissa SDK är nu i GA och ändringar i dokumentationen väntar. 

SDK: er för Bing Search göra sökfunktionen web åtkomlig i följande programmeringsspråk:

    * Se även [.NET-biblioteken](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingEntitySearch) för definitioner och beroenden.
* Kom igång med [Node.js-exempel](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Se även [Node.js-bibliotek](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/entitySearch) för definitioner och beroenden.
* Kom igång med [Java-exempel](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Se även [Java-bibliotek](https://github.com/Azure/azure-sdk-for-java/tree/master/azure-cognitiveservices/search/bingentitysearch) för definitioner och beroenden.
