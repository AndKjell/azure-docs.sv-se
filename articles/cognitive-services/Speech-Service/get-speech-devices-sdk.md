# <a name="get-the-cognitive-services-speech-devices-sdk"></a>Hämta kognitiva Services tal enheter SDK

## <a name="requesting-access"></a>Begär åtkomst till

Tal enheter SDK är i begränsat preview och du måste vara registrerade i programmet. För närvarande föredrar Microsoft stora företag som kandidater för åtkomst till den här produkten.

Följ dessa steg för att få åtkomst till tal enheter SDK:

1. Gå till Microsoft tal enheter SDK [registreringsformuläret](https://aka.ms/sdsdk-signup).
1. Läs den [licensavtalet](speech-devices-sdk-license.md).
1. Om du accepterar villkoren i licensavtalet, markera ”jag samtycker”.
1. Svara på frågorna i formuläret.
1. Skicka formuläret. 
1. Om din e-postadress inte redan är en del av Azure Active Directory får du ett e-postinbjudan se nedan vid godkännande. Om din e-postadress är redan i Azure Active Directory, du får ett e-postmeddelande från Microsoft Speech teamet vid godkännande och du kan gå vidare till [hämta tal enheter SDK](#download-the-speech-devices-sdk).

## <a name="approval-e-mail"></a>E-post för godkännande

```
From: Microsoft Speech Team from Microsoft (via Microsoft) <invites@microsoft.com> 
Subject: You're invited to the Microsoft organization 
```

![E-postmeddelande](media/speech-devices-sdk/get-sdk-1.png)

## <a name="accept-access"></a>Acceptera åtkomst
Utför stegen nedan för att ansluta till Azure Active Directory med den e-postadress som du angav under registreringen. Den här processen ger dig åtkomst till tal enheter SDK [hämtningsplats](https://shares.datatransfer.microsoft.com/).

1. Klicka på **Kom igång** i e-postmeddelandet du fick. Om din organisation redan är en Office 365-kund kan du uppmanas att logga in och kan gå vidare till steg 8.

2. Klicka på **nästa** i lanserade webbläsarfönstret.

    ![fönster för autentisering](media/speech-devices-sdk/get-sdk-2.png)

3. Skapa ett Microsoft-konto om du inte redan har ett. Ange samma e-postadressen som du har fått e-postinbjudan i steg 6 ovan.

    ![Skapa Microsoft-konto](media/speech-devices-sdk/get-sdk-3.png)

4. Klicka på **nästa** att skapa ett lösenord.

5. När du uppmanas att verifiera din e-post, tillbaka till din inkorg få verifieringskoden som skickas till dig.
 
7. Klistra in eller ange säkerhetskoden från e-postmeddelandet i dialogrutan. I det här exemplet är det ”8406”. Klicka sedan på **Nästa**.

    ![Kontrollera e-post](media/speech-devices-sdk/get-sdk-6.png)
 
8. Du har bekräftat att din e-postadress (från steg 6) är nu en del av Azure Active Directory när du ser programmet åtkomst panelen i webbläsarfönstret. Nu har du åtkomst till hämtningsplatsen tal enheter SDK.

## <a name="download-the-speech-devices-sdk"></a>Hämta tal enheter SDK

Gå till den [tal enheter SDK hämtningsplats](https://shares.datatransfer.microsoft.com/) och logga in med Account du skapade tidigare. Du kan hämta SDK för tal-enheter, associerade exempelkod och referensmaterialet genom att följa dessa steg.

![SDK-Hämtningsplats](media/speech-devices-sdk/get-sdk-7.png)

1. Hämta och installera Aspera Connect-verktyget när du uppmanas att göra det i webbläsaren.

    ![Hämta Aspera ansluta](media/speech-devices-sdk/get-sdk-8.png)
 
1. Klicka på **Ja** växla till Aspera ansluta.

    ![Växla till Aspera ansluta](media/speech-devices-sdk/get-sdk-9.png)
 
1. Klicka på **Tillåt** att bekräfta att hämta filer med Aspera Connect.

    ![Hämta med Aspera Connect](media/speech-devices-sdk/get-sdk-10.png)
 
1. Stäng fönstret för Aspera ansluta överföringar när filerna har hämtats.

    ![Aspera ansluta överföringar fönster](media/speech-devices-sdk/get-sdk-11.png)
 
Som standard filerna har hämtats till din **hämtar** mapp. Du kan logga ut från den här platsen nu.
