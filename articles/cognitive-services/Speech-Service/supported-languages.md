# <a name="supported-languages-for-speech-service"></a>Språk som stöds för tal-tjänsten

Olika språk stöds för olika funktioner för tal-tjänster. Tabellerna nedan sammanfattar språkstöd.

## <a name="speech-to-text"></a>Tal till text

Microsoft-taligenkänning API stöder följande språk. 

| Kod | Språk | Kod | Språk |
|-----|-----|-----|-----|
| ar T.ex | Arabiska (Egypten), moderna standard | IT-IT | Italienska (Italien) |
| de-DE | Tyska (Tyskland) | ja-JP | Japanska (Japan) |
| sv-SE | Engelska (USA) | pt-BR | Portugisiska (Brasilien) |
| es-ES | Spanska (Spanien) | ru-RU | Ryska (Ryssland) |
| fr-FR | Franska (Frankrike) | zh-CN | Kinesiska (Mandarin, förenklad) |

### <a name="customization"></a>Anpassning

Anpassad **Text till tal** röst teckensnitt stöder amerikansk engelska (en-US) och kinesiska (zh-CN).

## <a name="speech-translation"></a>Talöversättning

Den **tal översättning** API: et stöder olika språk för översättning tal till tal- och tal till text. Käll-språk måste alltid vara från tabellen talspråk. Tillgängliga mål språk beror på om översättning målet är tal eller text.

Du kan hämta en lista med språk antingen via programmering eller med hjälp av REST API-slutpunkten för språk. Listan innehåller varje språkkod, samt språknamnet på engelska, eller i andra språk som stöds. Den här listan uppdateras av tjänsten Microsoft Translator när ett nytt språk blir tillgängliga.
