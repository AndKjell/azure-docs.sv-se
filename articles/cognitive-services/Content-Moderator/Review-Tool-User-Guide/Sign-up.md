# <a name="sign-up-for-content-moderator"></a>Registrera dig för innehåll kontrollant #
Navigera till den [innehåll kontrollant registrering](http://contentmoderator.cognitive.microsoft.com/Account/Signup) sidan. Du kan logga med ditt Microsoft-konto eller skapa ett konto på webbplatsen innehåll kontrollanten.

![Registrera dig](images/0-signup-1.PNG)

## <a name="name-your-team"></a>Namnge ditt Team ##
Ange ett namn för din grupp för granskning. Alternativt kan erbjuda du dina kollegor så att ansluta till ditt team.

![Skapa Team](images/1-Team-Name.PNG)
