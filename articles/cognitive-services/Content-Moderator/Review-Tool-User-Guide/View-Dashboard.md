# <a name="view-dashboard"></a>Visa instrumentpanelen #

Klicka på den **instrumentpanelen** FLIKEN för att se nyckelvärden för alla innehållsgranskning gjort i verktyget. Du kan se Totalt antal slutförda och pågående omdömen för bilder och text. Du kan se mått för standard-teamet och din subteams.
