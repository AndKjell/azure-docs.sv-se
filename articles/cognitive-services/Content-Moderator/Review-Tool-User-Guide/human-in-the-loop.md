# <a name="human-in-the-loop"></a>HR-i-the-Loop

Du får bästa resultat när människor och maskiner arbeta tillsammans med innehållet måtta. Datorer utöka effektivt mänsklig granskningar där förutsägelse förtroende måste vara stödd eller härdat i ett verkligt sammanhang. Resultatet är en hybrid innehåll avbrottsmoderering process som presterar bättre än om människor eller datorer arbetade fristående.

## <a name="how-the-review-tool-helps"></a>Hur verktyget granska hjälper ##

Verktyget mänsklig granska när de används tillsammans med måtta datorn stödd API: er kan du utföra dessa viktiga uppgifter i förhållande till innehåll avbrottsmoderering livscykel.

1. Automatisera skapandet av mänsklig granskningar från underliggande avbrottsmoderering API-resultat
2. Använda ett verktyg (granska verktyget och API) att måttlig flera format (text, bild och video – kommer snart)
3. Tilldela eller eskalera innehåll granskningar till flera granska team ordnade efter innehåll kategori eller upplevelse nivå.
4. Använd standardarbetsflöden eller definiera anpassade arbetsflöden med flexibla regler och utan att skriva någon kod.
5. Lägg till mänskliga granska API eller business processen genom att helt enkelt skapa en koppling.
6. Använd standard kopplingar och granska resultatet från Microsoft PhotoDNA, Text Analytics och Ansikts-API: er.
7. Hämta nyckeltal på innehåll avbrottsmoderering-processer.
