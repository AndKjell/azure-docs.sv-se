# <a name="upload-images"></a>Ladda upp bilder #

Om du vill utforska användargränssnittet, överför du upp till fem bilder i taget, och varje avbildning högst 2 MB i storlek. Måste du kontrollera din filstorlekar innan du laddar upp.

![Ladda upp bilder](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>Skicka för Avbrottsmoderering ##
Om du måste överföra fler än fem bilder i taget, ta bort oönskad bilder med hjälp av ikonen Trashcan. Skicka avbildningar för automatisk måtta.

![Skicka bilder](images/1-try-image-2.png)
