# <a name="bing-visual-search-sdk"></a>Bing Visual Sök SDK

Bing Visual sökningen ger följande klient SDK: er som du kan använda för att utöva Visual Sök-API.

* [.NET-exempel](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [NuGet-paketet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.VisualSearch/1.2.0)
    * [.NET-biblioteken](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingVisualSearch) för definitioner och beroenden
* [Node.js-exempel](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * [Node.js-bibliotek](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) för definitioner och beroenden
* [Java-exempel](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * [Bibliotek för Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search) för definitioner och beroenden
* [Python-exempel](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * [Python-bibliotek](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) för definitioner och beroenden

SDK-exempel för varje språk som innehåller en Viktigt-fil med krav för att använda SDK tillsammans med anvisningar för att installera och köra exemplen.
