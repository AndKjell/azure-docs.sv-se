# <a name="knowledge-exploration-service"></a>Utforskningstjänst för Knowledge
Välkommen till tjänsten Microsoft Knowledge undersökning.  Knowledge utforskning Service (KES) erbjuder ett snabbt och effektivt sätt att lägga till interaktiva Sök och förfining i program.  Med KES, kan du skapa ett komprimerat index från strukturerade data, författare en grammatik som tolkar naturligt språk och tillhandahålla interaktiva frågan formulering med automatisk komplettering förslag.

Se [komma igång](GettingStarted.md) för en exempel-genomgång av hur du använder KES för att skapa en interaktiv sökning gränssnitt för en domän academic publikationer.

![AcademicAutoComplete](AutoComplete.png)

KES har flyttats från kognitiva förhandsversionen av tjänster till kognitiva Services övningar. Nya startsidan för projektet är [ http://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration ](http://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration).
