# <a name="how-to-sign-up-for-the-microsoft-translator-speech-api"></a>Hur du registrerar dig för Microsoft översättare tal-API

Följ dessa steg för att registrera dig för Microsoft översättare tal-API.

## <a name="1-sign-into-azure"></a>1: Logga in på Azure   
- Har du inte något konto? [Registrera dig för Microsoft Azure-konto](http://azure.com/)
- Redan har ett konto? [Logga in](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: prenumerera på Microsoft Translator     
- När du loggar in i Azure går du till den [kognitiva Services](https://portal.azure.com/#create/Microsoft.CognitiveServices) avsnitt
- Markera under ”API typ” översättare tal-API
- Markera den prisnivå som bäst passar dina behov under prisnivån
    - Varje prenumeration har en kostnadsfri nivå. Den kostnadsfria nivån har samma funktioner och funktionalitet som betald planer och ha inte ett förfallodatum.
    - Du kan ha endast en kostnadsfri prenumeration i ditt konto
- Fyll i resten av formuläret och klicka på knappen Skapa
- Prenumerationen är klar för omedelbar användning

## <a name="3-authentication-key"></a>3: autentiseringsnyckel    
- Hämta din autentiseringsnyckel genom att gå till menyn alla resurser > klickar du på din prenumeration
- Värdet för nyckeln används för autentisering, konfidentiell nyckel-värde. Du måste värdet för nyckeln när du utvecklar en app
