# <a name="custom-connectors-in-logic-apps"></a>Anpassade kopplingar i Logic Apps

Du kan skapa arbetsflöden och appar med [Azure Logic Apps](https://azure.microsoft.com/services/logic-apps), [Microsoft Flow](https://flow.microsoft.com), och [PowerApps](https://powerapps.microsoft.com) utan att behöva skriva någon kod. För att hjälpa dig integrera appar, data och affärsprocesser, erbjuda tjänsterna [~ 200 kopplingar](https://docs.microsoft.com/connectors/) - för Microsoft-tjänster och produkter, samt andra tjänster, till exempel GitHub, Salesforce, Twitter och mycket mer.

Men ibland kanske du vill anropa API:er, tjänster och system som inte finns tillgängliga som färdiga anslutningsappar. För att få stöd för mer skräddarsydda scenarier kan du skapa *anpassade anslutningsappar* med egna utlösare och åtgärder. Dokumentationsplatsen för kopplingar har slutförts grundläggande och avancerade kurs om anpassade kopplingar. Du kan starta med den [anpassad koppling översikt](https://docs.microsoft.com/connectors/custom-connectors/), men du kan också gå direkt till det här avsnittet för information om ett visst område:

* [Skapa en koppling för Logic Apps](https://docs.microsoft.com/connectors/custom-connectors/create-logic-apps-connector)

* [Skapa en anpassad anslutningsapp från en OpenAPI-definition](https://docs.microsoft.com/connectors/custom-connectors/define-openapi-definition)

* [Skapa en anslutningsapp från en Postman-samling](https://docs.microsoft.com/connectors/custom-connectors/define-postman-collection)

* [Använda en anpassad anslutningsapp från en logikapp](https://docs.microsoft.com/connectors/custom-connectors/use-custom-connector-logic-apps)

* [Dela anpassade kopplingar i din organisation](https://docs.microsoft.com/connectors/custom-connectors/share)

* [Skicka dina kontakter för Microsoft-certifikat](https://docs.microsoft.com/connectors/custom-connectors/submit-certification)

* [Vanliga frågor och svar för anpassade anslutningsappar](https://docs.microsoft.com/connectors/custom-connectors/faq)
