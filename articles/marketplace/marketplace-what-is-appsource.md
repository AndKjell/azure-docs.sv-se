# <a name="what-is-appsource"></a>Vad är AppSource
AppSource kan Microsoft-partner göra sina teknik och tjänster synlig för kunder via en stöds av Microsoft online-butik. Som en AppSource-utgivare kan du engagera användare i verksamheten på AppSource för att hitta, prova och få line-of-business SaaS-program samt implementeringstjänster för att Driv affärsresultat och minska tiden till värde: 

| Kunden behöver       | AppSource          | 
|:---------------------------------------- |:----------------------------------------------------- |
| **Letar du efter affärslösningar som fungerar med Microsoft-produkter använder de redan**    | Gör att partner kan använda program från tredje part och tjänster för att utöka Microsofts molnbaserade program och tekniker. |
| **Möjlighet att enkelt hitta rätt rätt lösning eller implementering av tjänsten.**     | Innehåller en affär för att upptäcka, prova, och få program och tjänster, tillägg och många fler. |
| **Branschspecifika line-of-business-lösning för att lösa sina specifika affärsutmaningar**   | Innehåller klar slutpunkt till slutpunkt-lösningar för att hjälpa adress specifika krav i många branscher. |
| **Appar för att förbättra produktiviteten och effektiviteten affärsinsikter**       | Tillhandahåller appar för företag, inklusive kundtjänst, HR, åtgärder och många fler. |
| **Implementering av erfarna partner som hjälper att anpassa appar till deras unika situation**     | Tillhandahålla en katalog med consulting services-erbjudanden för lösningar som bygger på Dynamics 365, Power BI, PowerApps och 3 part appar som är tillgängliga på AppSource för att hitta konsulttjänster utformad för att ge förutsägbara resultat för företagsanvändare. |

## <a name="appsource-publishing"></a>AppSource-publicering
Via AppSource, kan du visa ett program eller en konsult erbjudande som hjälper dig att uppfylla kundernas behov när du använder Microsoft-produkter som Office 365, Dynamics 365, Power BI och Power Apps. I AppSource-marknadsplatsen finns följande erbjudanden: 
*   **Dynamics 365 för Finance and Operations**: en Företagsresursplanering lösningen (Företagsresursplanering). Differentierade lösningar är Enterprise för medelstora till stora företag och Business Edition för små och medelstora företag.
*   **Dynamics 365 för kundengagemang**: en kund relation (CRM) hanteringslösning som inkluderar försäljning, kundtjänst, Field Service och Project Service Automation-program.
*   **Dynamics NAV Managed Service**: Microsoft **Dynamics NAV** är en Företagsresursplanering (Företagsresursplanering) programsvit för medelstora organisationer. Tjänsten erbjuder särskilda funktioner för tillverkning, distribution, myndigheter, detaljhandel och andra branscher.
*   **Power BI**: en visualisering för företagsanalyslösning.
*   **Consulting erbjudanden**: Services tillhandahållna av Microsoft-partners för att hjälpa kunder förstå, prova och implementera specialiserade tekniklösningar.
*   **Cortana Intelligence**: The Cortana Intelligence Suite är en samling oberoende, men helt integrerad data och analytiska verktyg som erbjuds av Azure.
*   **Office 365**: en modern arbetsplats i molnet. Samarbeta utan kostnad med Microsoft Word, PowerPoint, Excel och OneNote.

## <a name="azure-active-directory-integration"></a>Azure Active Directory-integrering
Vissa AppSource storefront program krävs för att integrera med Azure Active Directory (Azure AD) som ska publiceras. Integrering med Azure AD är väl dokumenterat; Microsoft tillhandahåller flera SDK: er och ytterligare resurser för att uppfylla dina krav. 

| Namn på erbjudande | Kräver Azure AD-integrering | Anteckningar |
|:--- |:---:|:--- |
| Dynamics 365 for Operations | Nej | Ett erbjudande för licensbaserade kräver inte Azure AD-integrering |
| Dynamics 365 för finans | Nej | Ett erbjudande för licensbaserade kräver inte Azure AD-integrering |
| Dynamics 365 för kundengagemang | Nej | Ett erbjudande för licensbaserade kräver inte Azure AD-integrering |
| Dynamics NAV hanterad tjänst | Nej | Ett erbjudande för licensbaserade kräver inte Azure AD-integrering |
| Power BI | Nej | Ett erbjudande för licensbaserade kräver inte Azure AD-integrering |
| Consulting erbjudanden | Nej | Tjänster koordineras av partner, inte via ett webbgränssnitt |
| Cortana Intelligence | Ja | Azure AD är en bra idé att säkerställa en smidig kund erfarenhet av och enheter av högsta kvalitet leads |
| SaaS-appar (tidigare Webbappar) | Önskad | Azure AD är en bra idé att säkerställa en smidig kund erfarenhet av och enheter av högsta kvalitet leads |
