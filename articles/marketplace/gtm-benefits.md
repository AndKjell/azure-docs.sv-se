# <a name="gtm-benefits-on-the-marketplace"></a>GTM fördelar på marketplace   
Nya listor skyltfönster på marketplace är berättigad till ledigt Marketplace GTM fördelar. Efter lista kontaktar marknadsföring specialister på Microsoft du eller din marknadsföring kontakt rivstarta aktiviteter. Det finns inget som du kan utom för att kommunicera med Microsoft när du kontaktas.  

De aktiviteter som Microsoft tillhandahåller varierar beroende på din lösningsstatus på marketplace. Fördelar ökar för listor.  

| Storefront | Ytterligare fördelar som är tillgängliga  |  
|:--- |:--- |  
| AppSource | <table> <tr><td>Microsofts Gold partners</td></tr> <tr><td>Utvärderingsversion appar</td></tr> </table> |  
| Azure Marketplace | <table> <tr><td>Transact erbjudanden</td></tr> <tr><td>Utvärderingserbjudanden</td></tr> </table> |  

Om du vill maximera effekten av dessa aktiviteter uppmuntrar Microsoft har att implementera starta planen. Du kanske vill använda din Landningssida för många av Start-medvetande.  

>[!NOTE]
>En extern Partner katalog (OCP katalog) är en fördel med medlemskap i Microsoft Partner Network.  

![GTM fördelar på marketplace](./media/marketplace-publishers-guide/gtmmay2018.png)  

Gå till marknaden-support innehåller mallar, webbinnehåll, utbildning och verktyg för att utveckla ditt företag.  
*   Forfor finns mer information om stöd för gå till marknaden, Microsoft gå till marknaden Services sidan finns på [partner.microsoft.com/reach-customers/gtm](https://partner.microsoft.com/reach-customers/gtm).
