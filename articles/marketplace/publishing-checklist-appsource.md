# <a name="publishing-checklist-for-appsource"></a>Publishing checklista för AppSource  
Granska nödvändiga komponenter innan du startar publiceringsprocessen.  

Följande artefakter krävs för att publicera en app på AppSource.  

## <a name="checklist"></a>Checklista  

| Listtyp | Erbjudandetyp | Publishing artefakt |   
|:--- |:--- |:--- |  
| Visa lista | Konsulter | <table> <tr><th>Storefront information</th></tr> <tr><td>Erbjudandenamn (200 tecken)</td></tr> <tr><td>Beskrivning (2 000 tecken)</td></tr> <tr><td>MPN-ID</td></tr> <tr><td>Land/region tillgänglighet</td></tr> <tr><td>Tillämpliga branscher, kategorier och sökord</td></tr> <tr><td>Skärmbilder (minst 1 280 x 720; maximalt 5)</td></tr> <tr><td>Marknadsföring dokument (maximalt 3)</td></tr> <tr><td>Leda mål</td></tr> <tr><td>Logotyp (48 x 48, 216 x 216)</td></tr> <tr><td>Behörighet</td></tr> <tr><td>Produkt-översikten (valfritt)</td></tr> </table> |
