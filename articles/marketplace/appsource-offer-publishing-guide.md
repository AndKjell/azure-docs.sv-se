# <a name="office-365-dynamics-365-power-apps-and-power-bi-offer-publishing-guide"></a>Guide för Office 365, Dynamics 365, Powerapps och Power BI-erbjudandet publicering

AppSource-storefront innehåller erbjudanden som bygger på eller utöka Dynamics 365, Office 365, Power BI och Power Apps. Var och en av dessa produkter har specifika dokumentationen att hjälpa dig publiceringsprocessen. Nedan finns länkar till mer information för varje typ av erbjudande. 

AppSource har lista och testversioner med anrop till åtgärden ”kontakta mig” eller ”Prova nu”. Det finns för närvarande inga funktioner för transaktion i AppSource.

## <a name="product-specific-guides"></a>Produktspecifika guider

### <a name="office-365"></a>Office 365

Granska den [publicering processen och riktlinjer](https://docs.microsoft.com/en-us/office/dev/store/submit-to-the-office-store)

### <a name="power-apps"></a>Power-appar

Lär dig mer om hur kunder kan [provköra din app på AppSource.](https://powerapps.microsoft.com/en-us/blog/appsource-test-drive/)

### <a name="power-bi"></a>Power BI

Granska den [publicering processen och riktlinjer.](https://docs.microsoft.com/en-us/power-bi/developer/office-store)

### <a name="dynamics-365"></a>Dynamics 365

#### <a name="dynamics-365-for-finance-and-operations"></a>Dynamics 365 for Finance and Operations
När du skapar för Enterprise Edition, granska de [publicering processen och riktlinjer](https://docs.microsoft.com/en-us/dynamics365/unified-operations/dev-itpro/lcs-solutions/lcs-solutions-app-source)

### <a name="dynamics-365-for-customer-engagement"></a>Dynamics 365 för kundengagemang
Granska den [publicering processen och riktlinjer](https://docs.microsoft.com/en-us/dynamics365/customer-engagement/developer/publish-app-appsource)
