# <a name="add-bank-and-tax-information"></a>Lägga till information om bank och skatterna  
Bank-och moms krävs för alla kommersiella erbjudanden med hjälp av transact listtyp.  
*   Om du publicerar kommersiella erbjudanden för inköp sedan du måste lägga till beloppet och skatt och valideringen i Dev Center.  
    >[!IMPORTANT]
    >För kommersiella erbjudanden (transact), du måste slutföra bank-och moms innan du kan trycka på dina erbjudanden till produktionen.  

*   Om du är publicering endast ledigt eller BYOL erbjuder och du behöver inte lägga till information. Du kan lägga till informationen senare, men validering av skatteinformationen om tar en stund. Om du planerar att erbjuda kommersiella erbjudanden för inköp, bör du lägga till information så snart som möjligt.  

## <a name="add-bank-information"></a>Lägg till bankinformation  
1.  Logga in på Dev Center med ditt Microsoft-ID.  
    *   Besök sidan finns i Windows Dev Center för att logga in Dev Center [dev.windows.com](https://dev.windows.com).  
2.  Välj beloppet kontot på den vänstra menyn.
3.  På den `Choose payment method` väljer `Bank account` eller `PayPal`.  
    >[!IMPORTANT]
    >Om du har kommersiella erbjudanden som kunder inköp på Marketplace är beloppet-kontot för där du får beloppet för inköp.  

4.  Ange informationen om betalningsalternativ. När du har kontrollerat att informationen är korrekt, klicka på knappen Spara.  
    >[!IMPORTANT]
    >Om du uppdaterar eller ändra kontot beloppet måste du följa samma steg som ersätter den aktuella informationen med den nya informationen. Ändringar i ditt konto beloppet kan fördröja betalningar upp till en betalning cykel. Den här fördröjningen uppstår eftersom Microsoft måste bekräfta ändringen av tjänstkontot kan det här är samma som när du först ställa in beloppet-konto. Du får betald hela när ditt konto har verifierats. På grund av betalningar från den aktuella betalningen cykel läggs till nästa cykel.  

4.  Klicka på knappen Nästa.

## <a name="add-tax-information"></a>Lägga till skatteinformation om  
1.  Logga in på Dev Center med ditt Microsoft-ID.  
    *   Besök sidan finns i Windows Dev Center för att logga in Dev Center [dev.windows.com](https://dev.windows.com).  
2.  Välj skatt profil på den vänstra menyn. 
3.  På den `Set up your tax form` väljer du land eller region där du har permanenta land.  
    
    Välj land eller region där du håller primära medborgarskap.  
    
    Klicka på knappen Nästa. 
4.  Ange skatt-information.  
5.  Klicka på knappen Nästa.
