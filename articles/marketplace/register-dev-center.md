# <a name="register-in-dev-center"></a>Registrera i Dev Center  
För att kontrollera att du inte är inloggad i ett personligt webbläsare konto, öppna ett nytt Internet Explorer InPrivate- eller Chrome Incognito-webbläsarsession.  
1.  Registrera som en säljare i Dev Center.  
    *   För att logga in och registrera som en säljare, gå till Azure Dev Center-sidan finns på [dev.windows.com/registration?accountprogram=azure](http://dev.windows.com/registration?accountprogram=azure).  
2.  Slutför den `Help us protect your account` som verifierar din identitet med hjälp av ditt telefonnummer eller e-postadress.  

    ![Rutor phone information i Hjälp oss att skydda ditt konto-guiden](./media/marketplace-publishers-guide/registerdevcenteremail.png)  

3.  I den `Registration - Account Info` väljer du kontot land eller region från den *konto land/region* nedrullningsbara menyn.  

    ![Kontoinformation med för land/region](./media/marketplace-publishers-guide/devcenterregistrationaccountinfo.png)  
   
    >[!WARNING]
    >Sälja dina tjänster i Azure Marketplace, kontrollera att din registrerade entitet från en av de godkända *säljer från* länder. Begränsningen plats krävs beloppet och skatt skäl. 
    >*   Mer information finns i sidan finns på Azure Marketplace deltagande principer [azure.microsoft.com/support/legal/marketplace/participation-policies](https://azure.microsoft.com/support/legal/marketplace/participation-policies).  

4.  För *kontotyp*, Välj den `Company` knappen.  
    *   För mer information om kontotyper och för att underlätta vid bedömningen som är bäst för dig, besök kontotyper platser, och avgift finns på [docs.microsoft.com/windows/uwp/publish/account-types-locations-and-fees](https://docs.microsoft.com/windows/uwp/publish/account-types-locations-and-fees).  

    Klicka på knappen Nästa.  

5.  För *Publisher visningsnamn*, ange visningsnamn (vanligtvis namnet på ditt företag).  

    >[!NOTE]
    >Publisher visningsnamn som angetts i Dev Center visas inte i Azure Marketplace när erbjudandet anges, men du måste fylla i den här kryssrutan om du vill slutföra registreringen. 

6.  För *kontaktuppgifter*, ange informationen som krävs för verifiering för kontot. 

    >[!IMPORTANT]
    >Du måste ange korrekt kontaktinformation. Verifieringsprocessen använder din kontaktinformation för att godkänna ditt företag i Dev Center. 

1.  För *företagets godkännare*, ange kontaktinformation för godkännare. Godkännare kontrollerar att du har behörighet att skapa ett konto i Dev Center för din organisation.  

    Klicka på knappen Nästa.  

    ![”Konto info” sidan med rödmarkerade](./media/marketplace-publishers-guide/devcenterregistrationpayment.png)  

8.  I den `Registration - Payment` avsnitt, måste du ange din betalningsinformation betala för Dev Center-konto.  
    *   För *kampanjkoden*, anger du en kampanjkod som täcker kostnaden för registrering.  
    *   För *fakturering*, ange din kreditkortsinformation. PayPal informationen kan användas i stället för ett kreditkort stöds marknader.  

    Klicka på knappen Nästa.  

    ![Dev Center betalningsinformation](./media/marketplace-publishers-guide/devcenterregistrationpayment2.png)  

9.  I den `Registration - Review` avsnittet, granska din kontoinformation och bekräfta att allt är korrekt.  

    Läs villkoren i avtalet för Microsoft Azure Marketplace-utgivare.  

    Klicka på kryssrutan som anger att du har läst och accepterat villkoren.  
    
    Klicka på knappen Slutför skickas en e-bekräftelse.  
11. Välj nästa steg med hjälp av vilken faktureringsmodell som tillämpas för ditt erbjudande.  

    | Faktureringsmodell | Nästa steg |  
    |:--- |:--- |  
    | Kostnadsfri | Gå till molnet Partner-portalen.<ul> <li>Om du vill publicera på Azure Marketplace finns i molnet partnerportalen sidan finns på [cloudpartner.azure.com](https://cloudpartner.azure.com).</li> </ul> |  
    | Kommersiella (transact) | Uppdatera din kontoinformation.<ul> <li>Mer information om att lägga till kontoinformation finns i [att använda Lägg till bank och skatterna information](#how-to-add-bank-and-tax-information) avsnitt.</li> </ul> |  

    *   Ett exempel på en extern erbjudandet är ett VM-erbjudande med en varje timme fakturering modellen.

## <a name="issue-dev-center-registration"></a>Problem: Dev Center-registrering  
Följ dessa steg om du vill skicka ett supportärende om du har problem med registreringen Dev Center.  
1.  Gå till Dev Center-stöd.  
    *   Besök sidan av developer support finns i Windows om du vill få support [developer.microsoft.com/windows/support](https://developer.microsoft.com/windows/support).  
2.  Kontakta oss under Klicka på Skicka en incident. 

    ![Knappen ”Skicka en incident”](./media/marketplace-publishers-guide/devcentersubmitincident.png)  

    Välj `Help with Dev Center` från den *problemtyp* nedrullningsbara menyn.  

    Välj `Publish and manage apps` från den *kategori* nedrullningsbara menyn.  
    
    Klicka på knappen Start e-post.  
1.  På inloggningssidan, logga in med någon Microsoft-ID. Om du inte har ett Microsoft-ID, skapar du en.  
    *   Mer information om hur du skapar en Microsoft-ID finns i [riktlinjer för att skapa ett Microsoft-ID för att hantera ett konto i Azure Marketplace](#guidelines-for-creating-a-microsoft-id-to-manage-an-azure-marketplace-account) avsnitt.  
2.  Fyll i information om problemet.  
3.  Klicka på knappen Skicka för att skicka biljetten.
