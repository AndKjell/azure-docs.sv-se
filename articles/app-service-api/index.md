# <a name="app-service---api-apps-documentation"></a>App Service – API Apps-dokumentation

API Apps är en av fyra apptyper som erbjuds av Azure App Service. API-appar i Azure App Service innehåller funktioner som gör det enklare att utveckla, hantera och använda API:er i molnet och lokalt. Azure App Service är en helt hanterad plattform för webb-, mobil- och integreringsscenarier.
