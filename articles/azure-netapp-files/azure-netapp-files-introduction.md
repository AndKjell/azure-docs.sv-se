# <a name="introduction-to-azure-netapp-files"></a>Introduktion till Azure NetApp Files
Azure NetApp Files-tjänsten är en företagsanpassad, avgiftsbelagd fillagringstjänst med höga prestanda. Azure NetApp Files fungerar med alla typer av arbetsbelastning och är högtillgänglig. Du kan välja garanterade service- och prestandanivåer och ställa in ögonblicksbilder via tjänsten. 

I dokumentationen kring Azure NetApp Files finns instruktioner kring hur du skapar och hanterar volymer med hjälp av Azure NetApp Files. 

> [!IMPORTANT] 
> Alla aktiviteter som beskrivs i dokumentationen kräver att du är med i programmet för allmänt tillgängliga förhandsversioner och given åtkomst till Azure-resursprovidern för Microsoft.NetApp. Mer information om att gå med i programmet för allmänt tillgängliga förhandsversioner finns på [registreringssidan för allmänt tillgängliga förhandsversioner för Azure NetApp Files](https://aka.ms/nfspublicpreview).
