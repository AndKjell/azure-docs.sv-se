# <a name="preview-upcoming-changes-to-azure-application-insights"></a>Förhandsgranska kommande ändringar till Azure Application Insights 

Application Insights släpper ofta nya funktioner. Om du vill se förhandsgranskningar av dessa förbättringar kan du registrera dig på bladet förhandsversionen av Application Insights.  Utvecklingsteamet tillgängliggör förhandsversioner av nya funktioner på ett begränsat sätt innan du gör dem tillgängliga för alla användare. 

Följande bild illustrerar hur du ställer in dina inställningar för förhandsgranskning.

![Application Insights Preview-bladet](./media/app-insights-preview/preview.png)

## <a name="set-preferences"></a>Ange inställningar

Du kan välja bland följande alternativ när du ser förhandsversioner på bladet förhandsvisning.

- Alltid: Du ser förhandsversion inträffar så fort de blir tillgängliga.
- Auto: Du ser förhandsversion inträffar som Microsoft rekommenderar för ditt konto. 
- Aldrig: Visas bara förhandsversion upplevelser som du väljer.
