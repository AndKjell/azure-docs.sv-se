# <a name="using-chef-with-azure"></a>Med Azure Chef
[Chef](http://www.chef.io) är en kraftfull automatiseringsplattform som omvandlar virtuella infrastrukturen i Azure till kod. Chef automatiserar hur infrastrukturen konfigureras, distribueras och hanteras i nätverket, oavsett dess storlek.

Den här artikeln beskriver fördelarna med att använda Chef för att hantera Azure-infrastrukturen.

## <a name="automate-infrastructure-apps-and-compliance-with-one-platform"></a>Automatisera infrastruktur, appar och kompatibilitet med plattformar
Företag kräver hastighet, hastighet och säkerhet konkurrera i digital marketplace. Tillsammans Chef och Microsoft hjälper enskilda användare, grupper och företag att göra detta. Med en enda plattform, Chef automatisera kan automatisera och kontinuerligt leverera din infrastruktur, program och även kompatibilitet mellan Microsoft-egendom.

## <a name="test-drive-chef-automate-on-azure"></a>Test Chef automatisera på Azure
Stöds av Chef, den [Chef automatisera Azure Marketplace lösning](https://azuremarketplace.microsoft.com/en-us/marketplace/apps/chef-software.chef-automate) gör att du kan skapa, distribuera och hantera din infrastruktur och program tillsammans. Ett enda klick kommer du direkt tillgång till alla kommersiella funktioner som ingår i Chef automatisera; insyn slutpunkt till slutpunkt över din hela flottan, aktivera kontinuerlig kompatibilitet och hantera alla ändringar med en enhetlig arbetsflöde.
