# <a name="rest-api-specification-for-azure-front-door-service"></a>REST API-specifikationen för Azure ytterdörren Service
Azure ytterdörren-tjänsten kan du definiera, hantera och övervaka globala routning för din Internet-trafik genom att optimera för bästa prestanda och omedelbar global redundans för hög tillgänglighet. Med åtkomsten, kan du omvandla din globala (flera regioner) konsument- och företagsprogram i robust, högpresterande anpassade moderna program, API: er och innehåll som når en global publik med Azure.


Följande tabell innehåller en referens för REST API för Azure ytterdörren-tjänsten.

API-Version: 2018-08-01

| Åtgärden grupp | Beskrivning |
|-------------|------------|
| [Ytterdörren](https://github.com/Azure/azure-rest-api-specs/blob/master/specification/frontdoor/resource-manager/Microsoft.Network/preview/2018-08-01-preview/frontdoor.json) | Definierar hur du kan skapa och hantera din ytterdörren-konfiguration |
| [Web Application-brandväggsprincip](https://github.com/Azure/azure-rest-api-specs/blob/master/specification/frontdoor/resource-manager/Microsoft.Network/preview/2018-08-01-preview/webapplicationfirewall.json) | Definierar hur du hanterar web application-brandväggsprincip för ytterdörren |
