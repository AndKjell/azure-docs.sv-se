# Get-AzureRmFrontDoor
Hämta ytterdörren konfiguration

## SYNTAX

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESKRIVNING
Den **Get-AzureRmFrontDoor** cmdletGet hämtar alla befintliga första dörrar i den aktuella prenumerationen matchar angivna informationen.
