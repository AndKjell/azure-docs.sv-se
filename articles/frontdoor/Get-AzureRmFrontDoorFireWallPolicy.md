# Get-AzureRmFrontDoorFireWallPolicy

Hämta WAF-princip

## SYNTAX

```
Get-AzureRmFrontDoorFireWallPolicy -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESKRIVNING
Den **Get-AzureRmFrontDoorFireWallPolicy** cmdletGet hämtar WAF-princip i en resursgrupp under den aktuella prenumerationen

## EXEMPEL

### Exempel 1: Hämta en WAF-princip som heter $Name i $resourceGroup
