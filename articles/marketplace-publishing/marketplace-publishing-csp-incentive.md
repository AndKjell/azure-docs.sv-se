# <a name="view-and-change-your-cloud-solution-provider-reseller-incentive-in-the-azure-marketplace"></a>Visa och ändra din leverantör ”återförsäljare Morot” i Azure Marketplace
Vi kommer att Cloud Solution Providers (CSP) ”återförsäljare Morot” tillgängliga för Marketplace utgivare på 15 mars 2016.  Standardinställningen för det här stimulansordningen värdet är **Ja**.  Du kan välja att CEIP stimulansordningen programmet som en utgivare.  Den här artikeln visar hur du ändrar inställningen i profilen för utgivaren så att du kan välja bort.

Följ dessa steg om du vill visa eller ändra ”återförsäljare Morot”-status:

1. Logga in på ditt [Microsoft Azure Publishing konto](https://publish.windowsazure.com/workspace)

   ![Profilen landning skärmen][1]
2. Klicka på **utgivare**

   ![Profilen landning skärmen - utgivare länka][2]
3. Se information om CSP: er i din profil.  Det finns en **växla till säga Ja eller Nej** Standardstatusen är ”återförsäljare Morot” - ”Yes”.

   ![Profil - utgivare][3]
4. Ändra din ”återförsäljare Morot” status åtkomst till din profil Publisher och sedan välja och spara ditt val:

   * Välj **Ja** kan välja att ge 20 procent rabatt till CSP: er som återförsäljning program och tjänster som du har angett i Azure Marketplace.  Eller välj **nr** att välja bort stimulansordningen programmet.  I båda fallen får utgivaren samma beloppet.
   * När du gör ett val se till att du klickar på **spara** i det övre högra hörnet.

   ![Profil - utgivare - ändra rutor][4]
