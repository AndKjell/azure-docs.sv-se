# <a name="how-to-troubleshoot-common-azure-marketplace-publisher-issues"></a>Felsökning av vanliga problem med Azure Marketplace publisher
Den här artikeln förklarar hur du löser eller få support för några vanliga scenarier för Marketplace.

## <a name="credits-and-refunds"></a>Krediter och återbetalningar
I vissa fall kan behöva du begära att ett kreditkort eller en återbetalning för en kund.  Microsoft Support kan hjälpa dig att se till att kundens kredit och återbetalning begäranden betraktas och att ett svar tillhandahålls i vederbörlig.

Följande visar hur du begär en kreditkontroll eller återbetalning för en kund.

> [!NOTE]
> Om en kund eller Azure återförsäljare kontaktar du Microsoft för att begära en återbetalning eller kredit, gäller vi samma principer som vi använder för vårt tillhandahållande av Azure-tjänster. Om vår principer visar att en återbetalning av köpeskillingen eller kredit förfaller, proaktivt utfärdas (och du godkänner oss att utfärda å dina vägnar) återbetalning av köpeskillingen eller kredit till kunden eller Azure återförsäljare, i tillämpliga fall. Om vår principer inte anger att en återbetalning av köpeskillingen eller kredit förfaller, kan vi erbjuder kunden eller Azure återförsäljaren för att kontakta dig direkt.
>
>

  ![1_RequestRefund][1]

1. Navigate to the support pages here: [https://support.microsoft.com/en-us/getsupport?tenant=ClassicCommercial&locale=en-us&supportregion=en-us&pesid=15635&sd=&oaspworkflow=start_1.0.0.0&wf=0&ccsid=635562546535117884](https://support.microsoft.com/en-us/getsupport?tenant=ClassicCommercial&locale=en-us&supportregion=en-us&pesid=15635&sd=&oaspworkflow=start_1.0.0.0&wf=0&ccsid=635562546535117884).
2. I den **problemtyp** väljer **hantera kunder**.
3. I den **kategori** väljer **Godkänn eller begäran om en kund återbetalning**.
4. Klicka på **startbegäran**.

   ![2_RequestRefund][2]

1. Ge kontaktinformation.
2. I den **vad är problemet?** anger du följande information:

   * Prenumerations-ID för påverkas kunden
   * Kundens namn
   * Hur mycket kredit ska vara för
   * Perioden för krediten
   * Faktura efter denna period
   * Orsaken till krediten

Du får sedan ett e-postmeddelande med bekräftelse inom 24 timmar att vi har tagit emot din information. När ett beslut har gjorts i förhållande till din begäran, informeras om resultatet.

## <a name="customer-subscription-validation"></a>Prenumerationen validering
I vissa fall kan du behöva Validera prenumerationsinformation för en av dina kunder. Du kan kontakta Microsoft Support att hjälpa dig med detta. Microsoft Support hjälper dig för att bekräfta att ett prenumerations-ID och faktureringsadress tillhör en känd kund.

  ![subscriptionvalidation][3]

* En kund kan kontakta dig kontaktar supporten.
* Du vet inte hur du verifierar att en kund är en äkta kunden för ett erbjudande.
* Du kan logga en supportförfrågan för att begära att kundens prenumeration.
* Du måste ange ett kund prenumerations-ID och faktureringsadress.
* Microsoft kontaktar dig för att bekräfta att faktureringsadress och prenumerations-ID är kända för oss.

## <a name="how-do-i-add-or-delete-an-administrator-from-my-publishing-account"></a>Hur lägga till eller ta bort en administratör från mitt publishing konto?
![manageadmins][8]

1. Navigera till publicering Portal.
2. Välj **utgivare**.
3. Välj **administratörer**.
4. Lägg till den medadministratör som du vill lägga till eller ta bort e-postadress.
5. Välj **Spara**.
