# <a name="technical-prerequisites-for-creating-a-solution-template-for-the-azure-marketplace"></a>Tekniska krav för att skapa en lösningsmall för Azure Marketplace
Läs noggrant innan du börjar processen och förstå varför och där varje steg utförs. Så mycket som möjligt du ska förbereda företagets information och annan information, hämta nödvändiga verktyg och skapa tekniska komponenter innan du börjar skapa erbjudande.  

## <a name="developing-building-blocks"></a>Utveckla byggblock
Innan du fortsätter med din lösningsmall på Publishing Portal måste du bestämma på byggblocken i din lösning. Nedan hittar du länkar till dokument som kan hjälpa dig i processen:

1. [Skapa Azure Resource Manager-mallar](../azure-resource-manager/resource-group-authoring-templates.md)
2. Azure Quickstart-mallar:
   * [https://azure.microsoft.com/documentation/templates/](https://azure.microsoft.com/documentation/templates/)
   * [https://github.com/azure/azure-quickstart-templates](https://github.com/azure/azure-quickstart-templates)

## <a name="next-steps"></a>Nästa steg
Nu när du har granskat förutsättningarna och slutföra de nödvändiga uppgifterna, du kan gå vidare med att skapa lösningen mallen erbjudandet enligt anvisningarna i den [Guide för att skapa en lösningsmall](marketplace-publishing-solution-template-creation.md). Om du vill återgå till huvudsidan för utfärdaruppgifter finns [komma igång: hur du publicerar ett erbjudande på Azure Marketplace](marketplace-publishing-getting-started.md).
