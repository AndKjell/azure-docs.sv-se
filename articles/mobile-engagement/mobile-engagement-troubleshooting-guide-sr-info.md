# <a name="service-request-sr-information"></a>Information om tjänstbegäran (SR)
> [!IMPORTANT]
> Azure Mobile Engagement upphör på 3/31/2018. Den här sidan tas bort strax efter.
> 

Ange följande information när du öppnar en tjänstbegäran för Mobile Engagement tjänster:

## <a name="ids-applicable-identifiers-related-to-your-issue"></a>ID: N: Tillämpliga identifierare relaterat till problemet:
* App-ID
* Kampanj-ID
* Enhets-ID
* Användar-ID
* Användarnamn
* Appinfo-tagg

## <a name="errors-applicable-error-information"></a>Fel: Tillämpliga felinformation
* Namnet på avsnittet API eller UI där problemet inträffar
* Text för felmeddelande som visas
* Resultatet av alla tester som du har utfört från den [felsökning guider](http://go.microsoft.com/fwlink/?LinkId=524382)

## <a name="code-applicable-coding-information"></a>Kod: Tillämpliga kodning information
* SDK-version och plattformen för din app (Android SDK 2.4.1, iOS 1.16.2 osv.)
* Hämta platsen för din produktionsapp (eller APK/TGZ-filer för utveckling av appen)
* ”AndroidManifest.xml” och/eller alla kodstycke från din app som är relaterade till Azure Mobile Engagement (för avancerad felsökning)
