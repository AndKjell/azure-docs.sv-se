# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Riktlinjer för Azure Data Lake Storage Gen1 kunderna

Azure Data Lake Storage Gen2 förhandsversion erbjudanden inbyggda Blob storage-funktioner, inklusive data lagringsnivåer, livscykelhantering och förbättrad nätverkssäkerhet, förutom Hadoop-kompatibla filsystemet tillgängligt i Data Lake Storage Gen1.

Om du är en befintlig Data Lake Storage Gen1-kund kan Microsoft uppmuntrar dig att utvärdera alla nya och förbättrade funktioner för Data Lake Storage Gen2 och ge feedback med [delta i förhandsversionen av](https://aka.ms/adlsgen2signup). Data Lake Storage Gen1 fortsätter även att stödjas. Data Lake Storage Gen1 och Gen2 kan samexistera, så dina befintliga resurser och program som utnyttjar Data Lake Storage Gen1 fortsätter att fungera utan uppdateringar. När du vill kan du utveckla en migreringsplan till Data Lake Storage Gen2.

Om du utvärderar för närvarande Data Lake Storage Gen1, rekommenderar Microsoft att du kan delta i förhandsversionen av Gen2 och utvärdera de nya och förbättrade funktioner som erbjuds av Data Lake Storage Gen2. På samma sätt, om du är en befintlig Blob storage-kund kan Microsoft uppmuntrar dig att utvärdera om ett Hadoop-kompatibla filsystem som passar behoven analytiska.
