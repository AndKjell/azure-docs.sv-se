# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Riktlinjer för Azure Data Lake lagring Gen1 kunder

Azure Data Lake lagring Gen2 Preview erbjudanden interna Blob storage funktionalitet, inklusive data skiktning livscykelhantering och förbättrad nätverkssäkerhet, förutom i Data Lake lagring Gen1 Hadoop-kompatibel filsystemet.

Om du är en befintlig Data Lake lagring Gen1 kund Microsoft rekommenderar att du utvärdera alla nya och förbättrade funktioner i Data Lake lagring Gen2 och ge feedback genom [deltar i förhandsgranskningen](https://aka.ms/adlsgen2signup). Data Lake lagring Gen1 fortsätter också att stödjas. Data Lake lagring Gen1 och Gen2 kan samexistera, så att dina befintliga resurser och program som utnyttjar Data Lake lagring Gen1 fortsätter att fungera utan uppdateringar. Du kan utveckla en migreringsplan till Data Lake lagring Gen2 vid behov.

Om du utvärderar för närvarande Data Lake lagring Gen1, uppmuntrar Microsoft att delta i Gen2 preview och utvärdera nya och förbättrade funktioner som erbjuds av Data Lake lagring Gen2. På samma sätt om du är en befintlig Blob storage kund uppmanar Microsoft dig att utvärdera om ett filsystem i Hadoop-kompatibel är lämpligt för dina analytiska behov.
