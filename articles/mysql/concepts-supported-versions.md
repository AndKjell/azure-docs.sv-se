# <a name="supported-azure-database-for-mysql-server-versions"></a>Azure-databas som stöds för MySQL server-versioner
Azure MySQL-databas har utvecklats från MySQL Community Edition [MySQL Community Edition](https://www.mysql.com/products/community/), InnoDB-motorn.  Azure-databas för MySQL stöder för närvarande följande versioner.

## <a name="mysql-version-5635"></a>MySQL-Version 5.6.35
Referera till MySQL [dokumentationen] (https://dev.mysql.com/doc/relnotes/mysql/5.6/en/news-5-6-35.html) mer information om förbättringar och korrigeringar i MySQL 5.6.35.

## <a name="mysql-version-5718"></a>MySQL-Version 5.7.18
Referera till MySQL [dokumentationen] (https://dev.mysql.com/doc/relnotes/mysql/5.7/en/news-5-7-18.html) mer information om förbättringar och korrigeringar i MySQL 5.7.18.

## <a name="managing-updates-and-upgrades"></a>Hantera uppdateringar och uppgraderingar
Azure-databas för MySQL hanterar automatiskt korrigering för delversion uppdateringar. För närvarande i förhandsversion stöds huvudversion uppgradering inte. Till exempel stöds uppgradera från MySQL 5.6 till MySQL 5.7 inte.
