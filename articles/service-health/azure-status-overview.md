# <a name="azure-status"></a>Azure-status
[Azure status](https://azure.microsoft.com/status/) ger dig en global vy över hälsotillståndet för Azure-tjänster. Du kan få aktuell information på tjänstetillgänglighet med Azure status. Alla har åtkomst till Azure status och kan visa alla tjänster som rapporterar sina tjänstens hälsa. 

![Status för Azure](./media/azure-status-overview/azure-status.PNG)

## <a name="azure-status-updates"></a>Uppdateringar av Azure
Sidan Azure status uppdateras i realtid som hälsa av ändring av Azure-tjänster. Om du lämnar sidan Azure status öppet, kan du styra den hastighet som uppdaterar sidan med nya data. Du kan se sidan uppdaterades senast överst.

![Uppdatera status för Azure](./media/azure-status-overview/update.PNG)

## <a name="azure-status-history"></a>Azure-statushistorik
Medan Azure statussidan visas alltid den senaste informationen om hälsa, kan du visa äldre händelser med hjälp av den [Azure status historiksidan](https://azure.microsoft.com/status/history/).

Det kan du visa händelser för upp till 90 dagar bakåt i tiden, inklusive preliminär grundorsaken, lösning och nästa steg för Azure.

## <a name="rss-feed"></a>RSS-Feed
Azure status innehåller också [en RSS-feed](https://azure.microsoft.com/status/feed/) av ändringar av hälsotillståndet för Azure-tjänster som du kan prenumerera på.
