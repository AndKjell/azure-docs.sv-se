# <a name="version-release-history"></a>Versionshistorik
Microsoft Genomics-teamet uppdaterar regelbundet Microsoft Genomics Python-klienten för korrigeringar och nya funktioner. 

## <a name="latest-release"></a>Senaste versionen
Den aktuella Python-klienten är version 0.7.4. Den släpptes mars 5 2018 och använder GATK 3.3. 


## <a name="release-history"></a>Versionshistorik 
Nya versioner av Microsoft Genomics Python-klienten släpps om två gånger per år. När nya versioner av Microsoft Genomics Python-klienten blir tillgängliga, uppdateras en lista med korrigeringar och funktioner här. När nya versioner släpps bör tidigare versioner fortsätta att ha stöd för minst 90 dagar. När tidigare versioner stöds inte längre, kommer att visas på den här sidan. 

### <a name="version-074"></a>Version 0.7.4
Version 0.7.4 har stöd för att acceptera SAS-token istället för kontonycklar i den `config.txt` indata. Mer information finns i [indata SAS-token Snabbstart](quickstart-input-sas.md). 

### <a name="version-073"></a>Version 0.7.3
Versionen 0.7.3 innehåller mindre felkorrigeringar

### <a name="version-072"></a>Version 0.7.2
Version 0.7.2 är den första versionen. Den 1 November-2017 släpptes och använder GATK 3.3.
