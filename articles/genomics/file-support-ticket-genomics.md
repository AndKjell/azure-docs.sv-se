# <a name="how-to-contact-microsoft-genomics-for-support"></a>Så här kontaktar du Microsoft Genomics för support
Den här översikten beskriver hur du en supportbegäran för att kontakta Microsoft Genomics-filen. Det kan vara användbart om du inte kunna felsöka problem med hjälp av den [felsökningsguide för](troubleshooting-guide-genomics.md) eller [vanliga frågor och svar](frequently-asked-questions-genomics.md). 


## <a name="file-a-support-ticket-through-the-azure-portal"></a>Filen ett supportärende via Azure portal
Först logga in på ditt Azure-konto i Azure-portalen. Klicka på frågetecken i det övre högra hörnet och välj `Help + support`.

![Kontakta supporten på Azure-portalen](./media/file-support-ticket/genomics-contact-support.png "kontakta supporten på Azure-portalen") 



Markera `New support request`. 

![Ny supportbegäran](./media/file-support-ticket/new-support-request.png "ny supportbegäran") 

Fyll i den `Basics`, inklusive `Issue type` och `Subscription`. Välj de mest relevanta `Issue type` problemet, som exempel i bilden nedan, `Technical` har valts. Se till att välja `Microsoft Genomics` som den aktuella tjänsten.  `Microsoft Genomics` kommer att visas under den `Intelligence and Analytics` kategori.   När du har valt `Microsoft Genomics`, du blir också ombedd att ange din `Support plan` typen.

![Stöd för begäran grunderna](./media/file-support-ticket/support-request-basics.png "stöd begäran grunderna")


Lägg till information om problemet. Verktyget begäran kommer också att länka till befintliga dokumentationen som kan vara bra att hjälpa dig. I det här exemplet problemet rör `Genomics keys`, men det finns också flera andra alternativ.

![Stöd för begäran problem](./media/file-support-ticket/support-request-problem.png "stöd begäran problem")

Senast, Lägg till din kontaktinformation och välj `Create` längst ned på skärmen för att skicka din supportbegäran.

![Stöd för begäran Kontakta](./media/file-support-ticket/support-request-contact.png "supportkontakten för begäran")

## <a name="next-steps"></a>Nästa steg
