# <a name="supported-browsers-and-devices-for-the-azure-portal"></a>Webbläsare som stöds och enheter för Azure-portalen
Du kan köra den [Azure-portalen](https://portal.azure.com) på skrivbordet för alla moderna Tablet PC-enheter och webbläsare.

### <a name="supported-devices"></a>Enheter som stöds
Azure-portalen körs bra på moderna datorer, Mac-datorer och surfplattor. Om du behöver hantera Azure-resurser från en mobil enhet kan du prova Azure mobilappen som är tillgänglig på iPhone- och Android. Mer information finns i [Azure mobilappar](https://azure.microsoft.com/features/azure-portal/mobile-app/).

### <a name="supported-browsers"></a>Webbläsare som stöds
Vi rekommenderar att du använder den senaste webbläsare som är kompatibel med operativsystemet. Följande webbläsare stöds:

* Microsoft Edge (senaste versionen)
* Internet Explorer 11
* Safari (senaste versionen, endast Mac)
* Chrome (senaste versionen)
* Firefox (senaste versionen)
