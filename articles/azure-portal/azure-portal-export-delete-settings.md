# <a name="export-or-delete-settings-in-the-azure-portal"></a>Exportera eller ta bort inställningarna i Azure-portalen
Du kan exportera och ta bort följande inställningar på menyn Azure portal inställningar:
* Azure portal privata instrumentpaneler
* Användarinställningar som favorit prenumerationer eller kataloger och senast loggade in directory
* Teman och anpassningar

Vi rekommenderar att du exportera och granska dina inställningar innan du tar bort dem. Det kan ta lång tid att återskapa instrumentpanelerna och lägga till dina anpassningar.

[!INCLUDE [GDPR-related guidance](../../includes/gdpr-intro-sentence.md)]

**Exportera eller ta bort dina portalinställningar**

1. Logga in på [Azure-portalen](http://portal.azure.com).
2. Högst upp i portalen, Välj **inställningar**.

    ![Skärmbild som visar den portalinställningar växeln](media/azure-portal-export-delete-settings/azure-portal-settings-icon.png)
3. Välj **exportera alla inställningar** eller **ta bort alla inställningar och privata instrumentpaneler**.

    ![Skärmbild som visar inställningar för export och ta bort](media/azure-portal-export-delete-settings/azure-portal-export-delete-settings.png)

      I följande tabell beskrivs de här inställningarna. 

      | Inställning | Beskrivning |
      | --- | --- |
      | **Exportera alla inställningar** | Skapar en JSON-fil som innehåller dina användarinställningar som din instrumentpanel tema.|
      | **Ta bort alla inställningar och privata instrumentpaneler** | Tar bort alla länkar till privata instrumentpaneler och andra användaranpassningar som du har gjort i portalens instrumentpanel. |
