# <a name="discover-jupyter-notebooks-in-the-azure-ai-gallery"></a>Identifiera Jupyter Notebooks i Azure AI-galleriet
[!INCLUDE [machine-learning-gallery-item-selector](../../../includes/machine-learning-gallery-item-selector.md)]

## <a name="jupyter-notebooks"></a>Jupyter Notebook
**[Jupyter-anteckningsböcker](https://gallery.cortanaintelligence.com/notebooks)**  innehåller koden, datavisualiseringar och dokumentation i en enda, interaktiva arbetsytan.
Datorer i galleriet ger självstudier och detaljerad information om avancerade maskininlärning tekniker och lösningar.

## <a name="discover"></a>Utforska
  Du kan bläddra för bärbara datorer i galleriet genom att öppna den [galleriet](http://gallery.cortanaintelligence.com) och klicka på **anteckningsböcker** överst på sidan Gallery.

 Den  **[Jupyter-anteckningsböcker](https://gallery.cortanaintelligence.com/notebooks)**  sidan visar en lista över de mest populära bärbara datorerna.
Klicka på **se alla** att visa alla bärbara datorer.
Den här sidan kan bläddra du alla anteckningsböcker i galleriet. Du kan också söka genom att välja filtervillkor till vänster på sidan och anger sökvillkoren längst upp.

 Klicka på alla bärbara datorer för att öppna den bärbara datorn informationssidan och läsa mer. Ge feedback på den här sidan som du kan kommentera eller ställa frågor till kommentarer. Du kan även dela den med vänner och kollegor med resurs-funktionerna i LinkedIn och Twitter. Du kan också e-en länk till anteckningsboken för att bjuda in andra användare att visa sidan.

![Dela objektet med vänner](./media/gallery-how-to-use-contribute-publish/share-links.png)

![Lägga till egna kommentarer](./media/gallery-how-to-use-contribute-publish/comments.png)

## <a name="download"></a>Ladda ned
Du kan hämta en kopia av alla bärbara datorer från galleriet på din arbetsyta i Machine Learning Studio.

### <a name="download-from-the-gallery"></a>Ladda ned från galleriet
Hämta en kopia av en bärbar dator från galleriet:

1. Öppna den bärbara datorn information i galleriet
2. Klicka på **öppna i Studio**
3. Välj den Region och arbetsytan som du vill använda
   
    ![Öppna Jupyter-anteckningsbok från galleriet](./media/gallery-jupyter-notebooks/open-notebook-from-gallery.png)

När du klickar på **öppna i Studio**, anteckningsboken läses in i Machine Learning Studio-arbetsytan och öppnas (om du inte redan är inloggad till Studio, uppmanas du att logga in med ditt Microsoft-konto innan den bärbara datorn kopieras till din arbetsyta). Du hittar den bärbara datorn igen senare genom att klicka på **anteckningsböcker** till vänster på startsidan Studio.

### <a name="download-in-machine-learning-studio"></a>Ladda ned i Machine Learning Studio
Du kan också öppna någon av samma galleriet bärbara datorer när du arbetar i Machine Learning Studio:

1. I Machine Learning Studio klickar du på **+ ny**
2. Välj **anteckningsboken** -visas en lista över datorer från galleriet att du kan välja bland eller du kan söka efter en specifik bärbar dator med hjälp av sökrutan
3. Peka med musen på en bärbar dator och väljer **Öppna anteckningsbok**
   
    ![Öppna galleriet anteckningsboken från i Machine Learning Studio](./media/gallery-jupyter-notebooks/open-notebook-from-studio.png)

En kopia av den bärbara datorn hämtas och öppnas i arbetsytan i avsnittet Jupyter Notebooks i Machine Learning Studio.
Anteckningsboken visas, tillsammans med dina anteckningsböcker på den **anteckningsböcker** sidan Studio (på den [Studio startsidan](https://studio.azureml.net/), klickar du på **anteckningsböcker** till vänster).
