# <a name="use-the-ai-toolkit-for-azure-iot-edge"></a>Använd AI Toolkit för Azure IoT kant

Styrkan styrs av datorn, machine learning och avancerade analyser kant. Du kan distribuera AI och maskininlärning där dina data finns i AI verktygssamlingen för Azure IoT kant. Driftsätta modellen som en dockerbehållare med en REST-API och trycker sedan modellen till en insticksenhet med Azure IoT-hubben. AI Toolkit för Azure IoT kant ger intelligence, elastisk beräknings- och kraften i molnet för att platser med begränsad, tillfälligt, eller ingen anslutning.

AI-verktygen för Azure IoT Edge är en samling av skript och kod distribueras behållare. Exempel inkluderar förutsägande underhåll, bild klassificering och taligenkänning samt anpassade modellen distribution via Azure Machine Learning och Azure IoT Hub. Modeller som ingår i verktygssamlingen kan användas som-är, men alla källkod och data är tillgängliga för utvecklare att anpassa utifrån deras behov.

AI-verktygen för Azure IoT kant offentliga GitHub-lagringsplatsen finns på [aka.ms/AI-toolkit](http://aka.ms/AI-toolkit).
