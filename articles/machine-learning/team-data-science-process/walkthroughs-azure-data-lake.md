# <a name="azure-data-lake-data-science-walkthroughs-using-u-sql"></a>Med hjälp av U-SQL Azure Data Lake datavetenskap genomgång

Dessa genomgång använder U-SQL med Azure Data Lake för förutsägelseanalys. De följer stegen som beskrivs i Team av vetenskapliga data. En översikt över Team av vetenskapliga data, se [datavetenskap Process](overview.md). En introduktion till Azure Data Lake finns [översikt över Azure Data Lake Store](../../data-lake-store/data-lake-store-overview.md).

Ytterligare datavetenskap genomgångar som kör Team datavetenskap Process grupperas efter den **plattform** som de använder. Se [genomgång köra Team av vetenskapliga data](walkthroughs.md) för en specificering av de här exemplen.


## <a name="predict-taxi-tips-using-u-sql-with-azure-data-lake"></a>Förutsäga taxi tips med Azure Data Lake U-SQL

Den [Använd Azure Data Lake för datavetenskap](data-lake-walkthrough.md) genomgången visar hur du använder Azure Data Lake för datagranskning och binär klassificering uppgifter på ett sampel från NYC Taxitransport dataset för att förutsäga om en kund betalar ett tips.
