# <a name="introduction-to-the-geo-artificial-intelligence-data-science-virtual-machine"></a>Introduktion till Geo artificiell Intelligence datavetenskap virtuell dator

Azure **Geo AI datavetenskap VM** (Geo-DSVM) som levererar geospatiala analytics funktioner från Microsofts datavetenskap VM. Mer specifikt den här virtuella datorn utökar AI och data vetenskap verktyg i den [datavetenskap VM](overview.md) genom att lägga till ESRI är marknadsledande [ArcGIS Pro](https://www.esri.com/arcgis/products/arcgis-pro/overview) geografisk Information System.

Geo AI datavetenskap VM är en produkt som erbjuder som uppstår till out-of-samarbete mellan ESRI och Microsoft. Resultatet är en kraftfull miljö som ger dig möjlighet att:

- Utforska data med populäraste AI och data NO-verktyg
- tillämpa statistiska analyser data med geografisk eller spatial aspekter
- Träna modeller för förutsägelseanalys med hjälp av machine learning och djup learning algoritmer
- Skapa omfattande visualiseringen

Den stöds på den Windows 2016 Data vetenskap virtuella datorn. Python och R gränssnitt till ArcGIS är också förkonfigurerade på Geo-DSVM som aktiverar Programmeringsåtkomst till geospatiala analytics i AI-program. Jupyter-anteckningsbok prover också som hjälper Kom igång med att skapa AI program infused med geospatial intelligence.


## <a name="why-geo-ai-data-science-vm"></a>Varför Geo AI datavetenskap VM? 

Integrera geografisk plats och plats information med dina data ger en kraftfull ny dimension för att förstå en värld allt instrumenterats med IOT-enheter. En mängd olika program finns i stort sett alla segment i civilrättsliga society om kommersiellt, statliga, academic eller inte för vinst aktiviteter. Geospatiala analytics ger robust verktyg för att samla in, hantera och analysera geografiska och plats-baserad data och kraftfull visualiseringen som kan aktivera unika insikter om betydelsen av sådana data. 

Den [Data vetenskap virtuell dator (DSVM)](overview.md) är populära experiment från Microsoft och modellering miljön i Azure, som ger en värd AI, Machine Learning och vetenskap verktyg. Dessa plattformar och verktyg är alla lätt förkonfigurerade som hjälper dig att vara produktiva omedelbart. 

Genom att kombinera datavetenskap VM förutsägande funktioner med skalbarhet i Azure-molnet och geospatiala analytics programvara från [ESRI](http://www.esri.com), dataanalytiker och geo-spatiala analytiker har nu en produktiv plattform från som du vill hämta nya intelligence från deras data och identifiera nya möjligheter.
