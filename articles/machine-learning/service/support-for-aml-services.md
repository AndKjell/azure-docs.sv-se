# <a name="get-support-and-training-for-azure-machine-learning-services"></a>Få support och utbildning för Azure Machine Learning

Den här artikeln innehåller information om hur du lär dig mer om Azure Machine Learning-tjänster, samt få support för dina problem och frågor. 

## <a name="learn-more-about-azure-machine-learning"></a>Läs mer om Azure Machine Learning

På [Azure Machine Learning Documentation Center](../desktop-workbench/overview-what-is-azure-ml.md) hittar du videokurser och instruktionsguider. Dessa stegvisa guider introducera tjänsterna och förklarar datalivscykeln. Vi lägger till nytt material på Machine Learning Center med jämna mellanrum. 

Dessutom finns [videor som du kan se mer](https://azure.microsoft.com/resources/videos/index/?services=machine-learning)

Du kan **skicka förfrågningar** för ytterligare material med hjälp av feedback-länken längst ned i artikeln.

## <a name="get-support-for-azure-machine-learning-services"></a>Få support för Azure Machine Learning

Du kan få support för Azure Machine Learning-tjänster på följande sätt:

+ **Teknisk support**: Kontakta Azures tekniska support genom att besöka [supporten](https://azure.microsoft.com/support/options/) och välja Machine Learning. 

+ **MSDN-användarforum**: Ställ frågor, besvara frågor, och ansluter till andra användare i den [Azure Machine Learning-tjänster Supportforum på MSDN](https://social.msdn.microsoft.com/Forums/windowsdesktop/home?forum=MachineLearning).

+ **Stackspill**: Gå till Azure Machine Learning-communityn på [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning) märkta med ”Azure-Machine Learning”.

+ **Dela produktförslag** och funktioner begäranden i vår [Feedbackkanal för Azure Machine Learning](https://feedback.azure.com/forums/257792-machine-learning), vilket är också tillgänglig via länken längst ned i varje artikel.
