# <a name="stream-analytics-resource-health-page-in-the-azure-portal"></a>Stream Analytics hälsa resurssida i Azure-portalen

Den **Resource health** i Azure-portalen visar hälsotillståndet för dina Azure Stream Analytics-jobb. Här visas också tillämplig riktlinjer som hjälper dig att felsöka problem. 

Den **Resource health** sidan är avsedd att hjälpa dig att ta reda på om roten för ett problem i programmet, eller om det beror på en händelse i Azure. Det kan hjälpa till att du vet när du vill öppna ett supportärende för att få hjälp.

## <a name="troubleshooting-workflow"></a>Felsökning av arbetsflöde

Om du vill se resurshälsa för Stream Analytics-jobb, Välj den **Resource health** sidan. 

![Stream Analytics felsökning flödet - åtkomst](media/stream-analytics-resource-health/stream-analytics-access-troubleshooting.png)

I följande flödesschema kan hjälpa dig fatta beslut om vad du vill söka efter och när du ska öppna ett supportärende.

![Strömma Analytics felsökning flöde – mappa](media/stream-analytics-resource-health/stream-analytics-troubleshooting-map.png)
