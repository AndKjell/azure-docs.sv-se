# <a name="create-list-or-delete-a-user-assigned-managed-service-identity-msi-using-the-azure-portal"></a>Skapa, visa eller ta bort en användare som tilldelats hanterad tjänstidentitet (MSI) med Azure portal

[!INCLUDE[preview-notice](~/includes/active-directory-msi-preview-notice-ua.md)]

Hanterad tjänstidentitet ger Azure-tjänster med en hanterad identitet i Azure Active Directory. Du kan använda den här identiteten för att autentisera till tjänster som stöder Azure AD-autentisering, utan att behöva autentiseringsuppgifter i din kod. 

För närvarande går inte att skapa, visa eller ta bort en användare som tilldelats hanterad identitet med hjälp av Azure portal.  Kolla igen senare.
