# <a name="reset-the-password-for-a-user-in-azure-active-directory"></a>Återställa lösenordet för en användare i Azure Active Directory

## <a name="how-to-reset-the-password-for-a-user"></a>Hur du återställer lösenordet för en användare
1. Logga in på den [Azure AD Admin Center](https://aad.portal.azure.com) med ett konto som är en global administratör för katalogen.
2. Välj **användare och grupper**.

   ![Öppna användarhantering](./media/active-directory-users-reset-password-azure-portal/create-users-user-management.png)
3. Välj **alla användare**.

   ![Öppna gruppen Alla användare](./media/active-directory-users-reset-password-azure-portal/create-users-open-users-blade.png)
1. Välj en användare i listan.
2. Välj **Översikt** för den valda användaren och sedan **Återställ lösenord** i kommandofältet.

    ![Att välja kommandot Återställ lösenord](./media/active-directory-users-reset-password-azure-portal/create-users-reset-password-command.png)
6. På **Återställ lösenord**väljer **Återställ lösenord**.
