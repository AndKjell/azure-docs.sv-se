# Verwalten von Besitzern einer Gruppe

## Zuweisen von Besitzern zu einer Gruppe

**So fügen Sie im Azure-Portal Gruppen Besitzer zu**

1. Klicken Sie in der Azure-Portal auf **Active Directory**, und klicken Sie dann auf den Namen des Verzeichnisses Ihrer Organisation.
2. Klicken Sie auf der **Gruppen** Registerkarte.
2. Auf der **Gruppen** Seite, klicken Sie auf den Namen der Gruppe, der Sie Besitzer hinzufügen möchten.
3. Klicken Sie auf der Seite, auf **Besitzer hinzufügen**.
4. Auf der **Besitzer hinzufügen** auf den Namen des Benutzers, der als Besitzer dieser Gruppe hinzufügen und sicherstellen, dass dieser Name hinzugefügt werden soll die **ausgewählte** Bereich.


**So entfernen Sie im Azure-Portal einen Besitzer aus einer Gruppe**

1. Klicken Sie in der Azure-Portal auf **Active Directory**, und klicken Sie dann auf den Namen des Verzeichnisses Ihrer Organisation.
2. Klicken Sie auf der **Gruppen** Registerkarte.
3. Auf der **Gruppen** auf den Namen der Gruppe, der Sie Besitzer entfernen möchten.
4. Klicken Sie auf der Seite, auf der **Besitzer** Registerkarte.
5. Klicken Sie auf der Seite, auf den Namen des Besitzers, den Sie verwenden möchten, aus dieser Gruppe entfernen, und klicken Sie dann auf **Entfernen**.
