# Verwalten von Kennwörtern in Azure Active Directory

Wenn Sie Administrator sind, können Sie im klassischen Azure-Portal das Kennwort eines Benutzers in Azure Active Directory (Azure AD) zurücksetzen. Klicken Sie auf den Namen des Verzeichnisses und dann auf der Seite Benutzer, klicken Sie auf den Namen des Benutzers und anschließend unten im Portal auf **Kennwort zurücksetzen**.

Im weiteren Verlauf dieses Themas werden die umfassenden Kennwortverwaltungsfunktionen behandelt, die Azure AD unterstützt, einschließlich:

- **Self-Service** können Endbenutzer oder Administratoren ihre abgelaufenen oder nicht abgelaufenen Kennwörter ändern, ohne einen Administrator oder Helpdesk um Unterstützung.
- **Self-Service** zurücksetzen kann Endbenutzer oder Administratoren ihre Kennwörter automatisch zurücksetzen, ohne einen Administrator oder Helpdesk um Unterstützung. Die Self-Service-Kennwortzurücksetzung erfordert Azure AD Premium oder Basic. Weitere Informationen finden Sie unter [Azure Active Directory-Editionen](active-directory-editions.md).
- **Zurücksetzen des Kennworts vom Administrator gestartete** kann ein Administrator eines Endbenutzers oder ein anderer Administrator das Kennwort aus dem Azure-Portal zurücksetzen.
- **Berichte zu kennwortverwaltungsaktivitäten** bieten Administratoren Einblicke in die kennwortzurücksetzung und-Registrierung Aktivitäten in ihrem Unternehmen ereignen.
- **Das Rückschreiben von Kennwörtern** ermöglicht die Verwaltung von lokalen Kennwörtern aus der Cloud, sodass alle oben genannten Szenarien von oder im Auftrag von ausgeführt werden können, Verbund- oder kennwortsynchronisierten Benutzern. Für das Rückschreiben von Kennwörtern ist Azure AD Premium erforderlich. Weitere Informationen finden Sie unter [Erste Schritte mit Azure Active Directory Premium](active-directory-get-started-premium.md).

> [AZURE.NOTE]
> Azure AD Premium steht Kunden in China über die internationale Instanz von Azure AD zur Verfügung. Azure AD Premium wird derzeit im von 21Vianet in China betriebenen Microsoft Azure-Dienst nicht unterstützt.
