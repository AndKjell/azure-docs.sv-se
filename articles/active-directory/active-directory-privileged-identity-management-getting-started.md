# Erste Schritte mit Azure Privileged Identity Management

## Starten der Erweiterung Privileged Identity Management

1.  Erstellen Sie ein Verzeichnis in Azure Active Directory.
2.  Fügen Sie Azure Active Directory Benutzer hinzu.
3.  Erstellen Sie im Verzeichnis einige globale Administratoren.
4.  Melden Sie sich beim <http://portal.azure.com> mit einem globalen Administratorkonto.
5.  Klicken Sie auf die **Benutzername** in der oberen linken Ecke des Portals.
6.  Wählen Sie die **Verzeichnis** Sie gerade erstellt haben.
7.  Klicken Sie auf das **Neu** Symbol in der linken Navigationsleiste.
8.  Wählen Sie **Sicherheit + Identität** im Menü erstellen.
9.  Wählen Sie **Azure AD Privileged Identitätsmanagement** aus der **Sicherheit + Identität** Menü.
10. Lassen Sie die **Pin im Startmenü das Kontrollkästchen** aktiviert, und klicken Sie dann auf die **Erstellen** Schaltfläche. Das Privileged Identity Management-Dashboard wird geöffnet.
