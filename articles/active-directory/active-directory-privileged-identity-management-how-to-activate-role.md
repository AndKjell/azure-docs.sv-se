# Azure Privileged Identity Management: Aktivieren oder Deaktivieren einer Rolle

## Aktivieren oder Deaktivieren einer Rolle

1. Melden Sie sich beim [http://portal.azure.com](http://portal.azure.com).
2. Führen Sie die Schritte [Erste Schritte mit Azure Privileged Identity Management](active-directory-privileged-identity-management-getting-started.md) Azure PIM im Portal-Dashboard zu platzieren.
3. Nachdem Sie die Schritte im Sicherheits-Assistenten abgeschlossen haben, wird das Hauptmenü von Azure PIM angezeigt.
4. Klicken Sie auf **Meine Rolle aktivieren**.
5. Eine Liste der Ihnen zugewiesenen Rollen wird angezeigt.
6. Klicken Sie auf die Rolle, die Sie aktivieren möchten.
7. Klicken Sie auf **aktivieren**. Die **rollenaktivierung anfordern** Blatt wird angezeigt.
8. Geben Sie im Textfeld den Grund für die Aktivierungsanforderung ein.
9. Klicken Sie auf **OK**.  Die Rolle wird jetzt aktiviert.
10. Nach Aktivierung eine Rolle können Sie eine Rolle auch deaktivieren, indem Sie auf **deaktivieren**.  Darüber hinaus kann die Rolle des Benutzers entfernt werden, mithilfe der Schritte im [Hinzufügen oder Entfernen einer Rolle](active-directory-privileged-identity-management-how-to-add-role-to-user.md).

- Weitere Informationen zur rollenaktivierung Einstellungen finden Sie unter [wie Rollenaktivierung konfigurieren](active-directory-privileged-identity-management-how-to-configure-role-activation.md)
- Weitere Informationen zur Sicherheit für rollenaktivierungseinstellungen Warnungen finden Sie unter [Vorgehensweise beim Konfigurieren von Sicherheitshinweisen](active-directory-privileged-identity-management-how-to-configure-security-alerts).
