# <a name="how-to-configure-an-application-proxy-application-to-use-pingaccess"></a>Så här konfigurerar du ett Application Proxy-program att använda PingAccess

Vår samarbete med PingAccess nu kan du utöka fördelarna med Application Proxy program med huvud-baserad autentisering. Om dina program inte använder huvuden, finns våra [enkel inloggning dokumentationen](manage-apps/application-proxy-configure-single-sign-on-with-kcd.md) mer information om andra alternativ.

## <a name="overview-of-steps-and-recommended-documents"></a>Översikt över stegen och rekommenderade dokument

Om du vill konfigurera ett program med PingAccess, finns det fyra steg:

1.  Konfigurera Application Proxy-kopplingar

2.  Skapa ett program med Azure AD Application Proxy

3.  & Hämta konfigurera PingAccess

4.  Konfigurera program för PingAccess

Mer information om var och en av de här stegen finns våra [enkel inloggning med rubriker dokumentation](manage-apps/application-proxy-configure-single-sign-on-with-ping-access.md).
