# <a name="add-or-change-work-information-for-a-user-in-azure-active-directory"></a>Lägga till eller ändra arbetsinformation för en användare i Azure Active Directory
Den här artikeln beskriver hur du lägger till eller ändrar arbetsinformation, till exempel telefonnummer eller avdelningsnamn för en användare i Azure Active Directory (AD Azure). Information om att lägga till nya användare i din organisation finns i [lägga till nya användare i Azure Active Directory](active-directory-users-create-external-azure-portal.md).

## <a name="to-change-work-information"></a>Så här ändrar du information om arbete
1. Logga in på den [Azure AD Admin Center](https://aad.portal.azure.com) med ett konto som är en global administratör för katalogen.
2. Välj **användare och grupper**.

   ![Öppna användarhantering](./media/active-directory-users-work-info-azure-portal/create-users-user-management.png)
3. Välj **alla användare**.

   ![Öppna gruppen Alla användare](./media/active-directory-users-work-info-azure-portal/create-users-open-users-blade.png)
4. Välj en användare i listan.
5. Den valda användaren väljer **arbete Info**.

    ![Öppna arbetsinformation](./media/active-directory-users-work-info-azure-portal/active-directory-create-users-work-info.png)
6. Lägg till eller ändra arbetsinformation. Markera i kommandofältet **spara**.
