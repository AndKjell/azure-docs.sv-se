# <a name="view-or-search-for-your-user-groups-in-azure-active-directory"></a>Visa eller söka efter användargrupper i Azure Active Directory
Den här artikeln beskriver hur du visar grupper i Azure Active Directory (AD Azure). En av funktionerna i användarhanteringen i Azure AD gör att du kan använda grupper för att utföra hanteringsuppgifter, till exempel tilldela licenser eller behörigheter till ett antal användare på samma gång.

## <a name="how-do-i-see-all-the-groups"></a>Hur gör jag för att se alla grupper?
1. Logga in på [Azure Portal](https://portal.azure.com) med ett konto som är en global administratör för katalogen.
2. Välj **Alla tjänster**, ange **Användare och grupper** i textrutan och tryck på **Retur**.

   ![Öppna användare och grupper](./media/active-directory-groups-view-azure-portal/search-user-management.png)
3. På bladet **Användare och grupper** väljer du **Alla grupper**.

   ![Öppna gruppbladet](./media/active-directory-groups-view-azure-portal/view-groups-blade.png)
4. På bladet **Användare och grupper – alla grupper** kan du lägga till eller ta bort visningskolumner, filtrera listan för att söka efter en grupp eller göra ändringar i grupper som du har tillräcklig behörighet för att ändra.
