# <a name="microsoft-authenticator-app"></a>Microsoft Authenticator-appen

>[!Important]
>Innehållet är avsett för användare. Om du är administratör hittar du mer information om hur du konfigurerar och hanterar en Azure Active Directory (Azure AD)-miljö i [dokumentationen om Azure Active Directory](https://docs.microsoft.com/azure/active-directory).

Microsoft Authenticator-appen gör det enklare att skydda dina konton via tvåstegsverifiering när du har angett ditt lösenord. Men visste du att den helt kan ersätta lösenordet för ditt personliga Microsoft-konto? Lär dig mer om Microsoft Authenticator-appen och hur du konfigurerar och använder den.
