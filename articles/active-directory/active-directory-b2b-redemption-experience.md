# <a name="azure-active-directory-b2b-collaboration-invitation-redemption"></a>Azure Active Directory B2B-samarbete inbjudan inlösning

## <a name="azure-ad-and-microsoft-account-users"></a>Azure AD och Microsoft-kontoanvändare
För användare med befintlig Azure AD-kontot eller Microsoft-konton är inlösning upplevelsen lika enkelt som att logga in.

## <a name="social-id-user-first-time-redemption"></a>Första gången inlösning för sociala användar-ID
Azure AD B2B-samarbete är det enkelt för alla e-postadress som ska användas för inlösning. Titta på inlösning-upplevelse när en icke-Microsoft e-postadress används för B2B-samarbete. Detta flöde inlösning är mer komplicerad, eftersom du kan behöva skapa ett konto vid tidpunkten för inlösning. Checka ut i följande video:
