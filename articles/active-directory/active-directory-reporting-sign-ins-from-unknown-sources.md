# Anmeldungen aus unbekannten Quellen
<p>Dieser Bericht zeigt Benutzer erfolgreich angemeldet haben Ihrem Verzeichnis, während ein Client-IP-Adresse zugewiesen, die von Microsoft als IP-Adresse (Tor IP) eines anonymen Proxys erkannt wurde. Diese Proxys werden häufig von Benutzern verwendet, die die IP-Adresse ihres Computers verbergen möchten, und sie können in böswilliger Absicht genutzt werden. </p><p> Die Ergebnisse in diesem Bericht geben an, wie oft sich ein Benutzer bei Ihrem Verzeichnis von dieser Adresse aus angemeldet hat, und weisen die IP-Adresse des Proxys aus.</p>
