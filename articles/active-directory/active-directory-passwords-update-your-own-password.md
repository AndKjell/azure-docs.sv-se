# Aktualisieren Ihres eigenen Kennworts

## Registrieren für die Kennwortzurücksetzung
Die schnellste Methode zur Registrierung für Zurücksetzen des Kennworts ist, fahren Sie mit http://aka.ms/ssprsetup.  

 1. Navigieren Sie zu http://aka.ms/ssprsetup.
 2. Geben Sie Ihren Benutzernamen und Ihr Kennwort ein.
 3. Wählen Sie eine Option für registrieren, indem Sie auf **jetzt einrichten**.  In diesem Fall zeige ich Ihnen, registrieren meiner **Telefon für Authentifizierung**.
 
    ![][011]
   
 4. Wählen Sie den Ländercode aus der Dropdownliste aus, und geben Sie Ihre **vollständige Telefonnummer + Vorwahl**.
 
    ![][012] 
    ![][013]
   
 5. Aktivieren Sie eines der **Text mir** oder **Anrufen** Optionen.  In diesem Fall ich wähle **Text mich**, sendet die für mein Handy einen 6-stelligen Code.  Warten Sie, bis der Code auf Ihrem Telefon ankommt.
 
    ![][014]
 
 6. Wenn Sie den Code erhalten haben, geben Sie ihn in das Eingabefeld ein und klicken auf „Überprüfen“.
 7. Wenn Sie sehen, **Danke**, das ist alles!  Jetzt können Sie verwenden, was bei der Registrierung zum Zurücksetzen Ihres Kennworts jederzeit, indem Sie auf https://passwordreset.microsoftonline.com.
 
    ![][015]

 >[AZURE.IMPORTANT] Wenn Ihr Administrator für mehr als eine Option registrieren kann, empfehlen wir dringend auch eine Option für die Sicherung registrieren, für den Fall, dass Sie Ihr Telefon oder der Zugriff auf Ihre e-Mail-Adresse verlieren.

## Ändern des Kennworts in Office 365
Führen Sie die folgenden Schritte aus, um das Kennwort für Ihr Geschäfts- oder Schulkonto in Office 365 zu ändern.  Wenn Sie Ihr Kennwort und möchten es zurücksetzen vergessen haben, führen Sie die Schritte [hier](#how-to-reset-your-password).

 1. Melden Sie sich mit Ihrem Geschäfts- oder Schulkonto bei Office 365 an.
 2. Wechseln Sie zu **Einstellungen** > **Office 365-Einstellungen** > **Kennwort** > **Kennwort ändern**.
 3. Geben Sie Ihr altes Kennwort ein, geben Sie Ihr neues Kennwort ein, und bestätigen Sie das neue Kennwort.
 4. Klicken Sie auf **Speichern**.

Weitere Informationen hierzu auf die [Office 365-dokumentationscenter](https://support.office.com/article/Change-my-password-in-Office-365-for-business-d1efbaee-63a7-4c08-ab1d-71bf932bbb5d).

## Ändern des Kennworts im Zugriffsbereich
Die folgenden Schritte ausführen, ändern Ihre Arbeit oder Schule Kennwort aus der [Zugriffsbereich](https://myapps.microsoft.com).  Wenn Sie Ihr Kennwort und möchten es zurücksetzen vergessen haben, führen Sie die Schritte [hier](#how-to-reset-your-password).

 1. Melden Sie sich mit Ihrer Arbeit https://myapps.microsoft.com oder schulkonto an.
 2. Klicken Sie auf die **Profil** Registerkarte.
 3. Klicken Sie auf die **Ändern des Kennworts** Kachel auf der rechten Seite des Bildschirms.
 4. Geben Sie Ihr altes Kennwort ein, geben Sie Ihr neues Kennwort ein, und bestätigen Sie das neue Kennwort.
 5. Klicken Sie auf **Speichern**.
 
 Gibt es ein Problem beim Ändern Ihres Kennworts?  Erfahren Sie mehr über [häufige Probleme und deren Lösung](#common-problems-and-their-solutions).

## Zurücksetzen des Kennworts


Führen Sie die folgenden Schritte aus, um das Kennwort für Ihr Geschäfts- oder Schulkonto in einem beliebigen Anmeldebildschirm für Geschäfts- oder Schulkonten zurückzusetzen.

 >[AZURE.IMPORTANT] Diese Funktion ist nur verfügbar, wenn Ihr Administrator er eingeschaltet ist.  Wenn es nicht aktiviert ist, weist eine Nachricht Sie darauf hin, dass dieses Feature für Ihr Konto nicht aktiviert ist.  Den Link "Wenden Sie sich an Ihren Administrator" können in diesem Fall mit Ihren Administrator erhalten, um Ihr Konto zu entsperren.<br><br>Wenn Ihr Administrator Sie für diese Funktion aktiviert hat, müssen Sie zunächst registrieren, bevor Sie ihn verwenden können.  Können Sie dies hier: http://aka.ms/ssprsetup.
 

 1. Auf einem arbeiten oder Schul-Konto auf der Seite, klicken Sie auf einer "kann nicht den Zugriff auf Ihr Konto?" oder "Kennwort vergessen?" Links, oder navigieren Sie direkt auf https://passwordreset.microsoftonline.com.
 
    ![][001]
 
 2. Auf der "wer Sie sind?" Seite, geben Sie Ihre Geschäfts- oder Schul-Konto-Id und nachweisen, dass Sie ein Roboter durch Übergeben der Captcha nicht.
 
    ![][002]
 
 3. Klicken Sie auf die Schaltfläche „Weiter“.
 4. Wählen Sie eine Option zum Zurücksetzen Ihres Kennworts.  Abhängig davon, wie Ihr Administrator das System konfiguriert hat, sehen Sie eine oder mehrere der folgenden Optionen:
     * **Meine alternative e-Mail-Adresse e-Mail** -sendet eine e-Mail mit einem 6-stelligen Code entweder die **Alternative e-Mail-Adresse** oder **Authentifizierung e-Mail** (Auswahl).
     * **Text mein Mobiltelefon** -Texte Ihr Telefon mit einem 6-stelligen Code entweder die **Mobiltelefon** oder **Authentifizierung e-Mail** (Auswahl).
     * **Mein Mobiltelefon anrufen** -Aufrufe Ihrer **Mobiltelefon** oder **Telefon für Authentifizierung** (Sie auswählen) – drücken Sie die *#* -Taste, um den Aufruf zu überprüfen.
     * **Meinen geschäftsanschluss anrufen** -Aufrufe Ihrer **Telefon (Büro)** -Drücken Sie die *#* -Taste, um den Aufruf zu überprüfen.
     * **Meine Sicherheitsfragen beantworten** -zeigt Ihre zuvor registrierten Fragen Sie zu.

    ![][003]
     
 5. Hier verwenden wir die Option „Textnachricht an mein Mobiltelefon senden“ als Beispiel.  Wenn Sie eine telefonbasierte Option verwenden, müssen Sie zuerst Ihre Telefonnummer bestätigen, bevor wir Ihnen eine SMS senden.   Geben Sie die vollständige Telefonnummer ein, und klicken Sie dann auf **Weiter** richtig überprüfen und senden Sie einen Text.
 
    ![][004]
 
 6. Achten Sie nach dem Empfang der SMS darauf, dass Sie den Verifizierungscode im Nachrichtentext verwenden und nicht die Nummer, von der der Code gesendet wurde.  Es kann einige Minuten dauern, bis die SMS bei Ihnen eintrifft – Gelegenheit für einen Kaffee.
 
    ![][009]
 
 8. Geben Sie nun den auf dem Telefon empfangenen Code in das Eingabefeld auf der Seite ein.
 
    ![][005]
 
 9. Ihr Administrator kann festlegen, dass ein zweiter Überprüfungsschritt erforderlich ist. In diesem Fall müssen Sie Schritt 4 wiederholen und dabei eine andere Option auswählen.
 10. Im Fenster "Neues Kennwort wählen", wählen Sie ein neues Kennwort bestätigen Sie Ihre Auswahl, und klicken Sie auf **Fertig stellen**.
 
    ![][006]
    ![][007]
  
 11. Sobald die Seite mit der Erfolgsmeldung angezeigt wird, ist alles bereit.  Sie können sich mit Ihrem neuen Kennwort anmelden.
 
    ![][008]
 
Gibt es ein Problem beim Zurücksetzen Ihres Kennworts?  Erfahren Sie mehr über [häufige Probleme und deren Lösung](#common-problems-and-their-solutions).
