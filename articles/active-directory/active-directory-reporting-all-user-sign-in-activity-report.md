# Azure Active Directory-Bericht „Alle Benutzeranmeldeaktivitäten“

*Diese Dokumentation ist Teil der [Azure Active Directory Reporting Guide](active-directory-reporting-guide.md).*

 > [AZURE.IMPORTANT]
 >
 - Der Bericht „Alle Benutzeranmeldeaktivitäten“ ist ein Premium-Bericht, der nur in der privaten Vorschau zur Verfügung steht. Um Zugriff auf den Bericht abzurufen, e-Mail-aadreportinghelp@microsoft.com mit Ihrem Domänennamen (z. B. **contoso.onmicrosoft.com**)


Der Bericht „Alle Benutzeranmeldeaktivitäten“ enthält alle Anmeldungen von allen Benutzern in Ihrem Verzeichnis mit Datum und Uhrzeit, Benutzerprinzipalname (UPN, User Principal Name), IP-Adresse und der Anwendung, bei der sich der Benutzer angemeldet hat.


 > [AZURE.IMPORTANT]
 >
 - Der Bericht „Alle Benutzeranmeldeaktivitäten“ ist derzeit nur über die Reporting-API verfügbar. Besuchen Sie [Erste Schritte mit der Reporting-API](active-directory-reporting-api-getting-started.md) um mehr zu erfahren.
