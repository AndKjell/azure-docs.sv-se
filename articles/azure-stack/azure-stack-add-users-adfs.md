# <a name="add-users-in-the-azure-stack-development-kit"></a>Lägg till användare i Azure-stacken Development Kit

*Gäller för: Azure stacken Development Kit*

Om du vill lägga till fler användare Development Kit-distributionen måste du lägga till dem till Azure-stacken Development Kit katalogen med hjälp av Microsoft Management Console från värddatorn Azure stacken.
1.  Öppna Microsoft Management Console på värddatorn för Azure-stacken.
2.  Klicka på **fil > Lägg till eller ta bort snapin-modulen**.
3.  Välj **Active Directory-användare och datorer** > **AzureStack.local** > **användare**.
4.  Klicka på **åtgärd** > **nya** > **användaren**.
5.  I dialogrutan Nytt objekt – användare och ange och bekräfta ett lösenord
6.  Klicka på **nästa** att slutföra värdena och klicka på Slutför för att skapa användaren.
