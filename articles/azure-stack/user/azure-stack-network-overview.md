# <a name="introduction-to-azure-stack-networking"></a>Introduktion till nätverksfunktioner i Azure Stack

*Gäller för: Azure Stack integrerat system och Azure-stacken Development Kit*

Azure-stacken innehåller en mängd olika nätverksfunktioner som kan användas tillsammans eller separat:

- **Anslutning mellan Azure Stack-resurser**  
    Anslut Azure-resurser tillsammans i en säker, privat virtuellt nätverk i molnet.
- **Internetanslutning**  
    Kommunikation till och från Azure-stacken resurser via Internet.
- **Lokala anslutningsmöjligheter**  
    Ansluta ett lokalt nätverk till Azure-stacken resurser via ett virtuellt privat nätverk (VPN) via Internet eller via en dedikerad anslutning till Azure-stacken.
- **Läsa in belastningsutjämning och trafik riktning**  
    Läsa in Utjämna trafiken till servrar på samma plats och trafiken till servrar på olika platser.
- **Säkerhet**  
    Filtrera nätverkstrafiken mellan undernät eller enskilda virtuella datorer (VM).
- **Routning**  
    Använd standardroutning eller helt styra routning mellan din Azure-stacken och lokala resurser.
- **Hanterbarhet**  
    Övervaka och hantera dina Azure-stacken nätverksresurser.
- **Verktyg för distribution och konfiguration**  
    Använd en webbaserad portal och plattformsoberoende kommandoradsverktyg för att distribuera och konfigurera nätverksresurser.
